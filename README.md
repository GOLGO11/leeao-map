## 本地预览

已提供一个零依赖静态服务器：

```powershell
node tools/static-server.mjs
```

然后打开：

`http://127.0.0.1:8000/`

## 数据出口

- 地点主表：`data/places.csv`
- 事件-地点关联表：`data/event_places.csv`
- 重新导出：

```powershell
node tools/export-geo-csv.mjs
```

CSV 文件带 UTF-8 BOM，方便在 Excel 中直接打开中文。

## 数据原则

1. 每条事件必须带来源文件、来源定位和证据摘录。
2. 地点坐标必须标注精度，不能把城市级坐标写成精确门牌。
3. 不确定信息保留 `needs_review` 或较低 `certainty`，下一轮用具体书目交叉验证。
4. 一本书一本书处理，先抽取，再复核，再进入地图展示。
5. 地图只收入重要事件；全文地名穷尽另行建表，不与主线地图混在一起。
