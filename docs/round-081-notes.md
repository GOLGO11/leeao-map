# 第八十一轮：《李敖笑傲江湖》初筛
处理时间：2026-05-09

来源范围：`《大李敖全集5.0》（wjm_tcy版）分章节/010.节目演讲类/001.李敖笑傲江湖/`

## 本轮原则

本书是电视节目文字稿，共124篇，空间线索密集但多数是节目中谈论的历史地名、新闻地点、人物旧事或举例材料。本轮只处理能说明李敖本人电视阶段的节目空间、签约过程和已知公共表达节点；不把节目中谈到的美国、北京、总统府、法院、台湾大学、阳明山等对象地点拆成李敖足迹。

## 新增地点

- `place-truth-news-network-taipei`：真相新闻网（台北，旧址待考）。第一集称真相新闻网请李敖做节目，双方从1995年1月13日来来饭店日本料理会谈开始，谈判到1995年10月20日签约，并达成《李敖笑傲江湖》节目合约书。第73集又称节目作满一年后，与真相新闻网重新订下一年合约。本书没有给出真相新闻网门牌或摄影棚位置，故按台北市级坐标处理并标记待考。

## 补强既有节点

- `event-1995-tv-xiaoaojianghu`：地点由泛化的 `place-taipei` 改为 `place-truth-news-network-taipei`，并补入第一集和第73集作为节目签约、开播与续约证据。事件摘要也补入1月13日来来饭店谈判、10月20日签约和节目作满一年续约三条线索。

## 暂缓候选

- 来来饭店日本料理：是节目谈判的起点，但本书只作签约过程背景；暂不单独建饭店点。
- 1995年9月1日台湾大学商研所黄国隆会面、9月27日台湾大学记者招待会、9月28日李敖与吕安妮记者招待会：与第一集主题有关，但地点细节不足，且当前更适合未来“吕安妮事件/教育争议”专题层。
- 第28集谈节目录影方式，提到三台摄影机和工作人员，但没有录影棚、电视台或地址，不能收窄到具体棚别。
- 第73集谈家中不出门、忠孝东路买手表等，属于生活闲谈或微地点，不入主图。
- 第115集称《笑傲江湖》已讲七百多场，并解释字幕与录影带价值，可补充节目规模，但不新增地点。

## 初筛结论

《李敖笑傲江湖》初筛完成。本轮不制造新的密集节目点，而是把既有电视阶段节点从“台北”提升为“真相新闻网（台北，旧址待考）”，使地图更像正式项目的数据层。后续复查重点是：真相新闻网旧址、录影棚位置、1995年10月30日开播证据与节目后期是否转移平台。
