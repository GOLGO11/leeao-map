# 第六十九轮：《李敖书笺集》初筛

处理时间：2026-05-08

来源范围：`《大李敖全集5.0》（wjm_tcy版）分章节/008.书信函件类/006.李敖书笺集/`

## 本轮结论

本书为1984年2月至1986年9月间书笺、公开信与政治论战材料。初筛原则仍是只收李敖本人居住基点、明确出版查禁现场与能补强主线的空间材料；政论对象、旧报刊引文、收信人所在地与日常行政短行暂不密集入图。

## 新增入图

- `event-1982-1985-lincoln-building-residence` / `place-taipei-lincoln-building-dunhua-610`：据 `029.苍蝇也没那么好拍的！.txt`，李敖户口1982年3月15日迁入敦化南路610号林肯大厦，1983年短迁乐利路林正杰处后又迁回，1985年8月19日迁出；该信又记录他对林肯大厦楼下私娼与顶楼违建问题的介入。本轮作为1980年代中期台北居住/户籍基点新增，坐标暂按敦化南路旧号门牌近似。
- `event-1984-xueying-bookstore-seizure` / `place-taipei-xueying-bookstore-roosevelt-341`：据 `010.捞过了界的扣押.txt`，1984年8月28日台北市政府便衣人员至罗斯福路三段341号学英书局，扣押《万岁评论丛书6》与《李敖千秋评论丛书35》，并出具取缔违禁出版品收据。本轮作为具体书局流通查禁现场新增。

## 补强既有
