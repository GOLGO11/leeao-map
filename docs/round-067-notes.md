# 第六十七轮：《李敖书札集》初筛

处理时间：2026-05-08

来源范围：`《大李敖全集5.0》（wjm_tcy版）分章节/008.书信函件类/005.李敖书札集/`

## 本轮结论

本轮新增两个事件、两个地点。一本书中政论对象地名很多，但真正适合主线地图的是两个硬锚点：1969年安坑监狱探雷震遭驱逐，以及1982年至1983年《千秋评论》在出版/装订/书报流通现场遭查扣。

## 新增入图

- `event-1969-ankeng-prison-lei-chen-visit`：1969年1月10日，李敖与雷美琳一同到安坑监狱探雷震，遭安坑监狱政工官驱逐，不准与雷震见面。新增地点 `place-xindian-military-prison-ankeng`，按国家人权记忆库所载新北市新店区莒光路42号定位。
- `event-1982-1983-qianqiu-raids-four-seasons`：1982年12月1日《千秋评论》第十六期《政治·女人·蛇》在四季出版公司、统一装订厂、台北联丰书报社和中盘商处遭查；1983年5月24日第二十二期在印刷厂遭查扣；1983年10月29日第二十六期在装订时遭抢，随后台北县及台北市双园区查扣九千余本。新增地点 `place-taipei-four-seasons-publishing-xinsheng-94`，按本地材料“台北市新生南路三段94号六楼四季公司”近似。

## 暂缓入图候选

- `001.致管区警察头子书.txt`：信义路派出所户口校正争议，能补强信义路长期户籍/软禁背景，但主线权重偏日常。
- `002.讨老婆的一个新害处.txt`：殷海光身后事涉及台大医院、怀恩堂、市立殡仪馆、台大图书馆和玉成街等，材料价值高，但更适合殷海光专题复查。
- `008.给黄少谷先生的又一公开信.txt`：最高法院平反、天母静庐、敦化南路496号12楼地址线索重要，但多数是法律文书和判决对象空间，暂不新增主线点。
- `027.给方素敏的一封信.txt`：林义雄家属回台、医院与机场风险多为他人行动。
- `032.上县太爷书.txt`：彰化基督教医院、鹿港分局草港派出所为曾发粦案材料，不是李敖本人足迹。

## 复查提示

下一轮复查重点：

1. 安坑监狱点位是否需要补入更多雷震出狱线索，尤其1970年9月3日李敖约外国记者接雷震出狱。
2. 四季出版公司是否能交叉到更多地址/负责人资料；统一装订厂、台北联丰书报社、桂冠图书公司是否有可靠旧址。
3. 殷海光后事线是否值得在专题层新增台大医院/怀恩堂组合节点。
