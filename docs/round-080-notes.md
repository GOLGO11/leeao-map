# 第八十轮：《给马戈的五十封信》第一轮复查
处理时间：2026-05-09

来源范围：`《大李敖全集5.0》（wjm_tcy版）分章节/008.书信函件类/011.给马戈的五十封信/`

## 复查原则

本轮回看第七十九轮暂缓候选，重点区分“李敖本人行动轨迹”与“信中转述、他人地址、生活微点”。复查后只提升一条具有连续行程、明确日期与两个机构地点的台中短行；其余台大医院、木栅、台大分站、云和街十号等线索继续留作候选或背景。

## 新增入图

- `event-1958-taichung-agricultural-college-tunghai-visit`：据 `019.小儿女事转播.txt`，李敖1958年9月中旬抵台中后，赴张世民处谈至傍晚，次日上午到台中农学院看李善培，午后赴东海大学看李华俊，并在台中市面为马戈选购照片。`022.桃花路上.txt` 与 `023.寡情如此.txt` 又补明他预计9月23日清晨北上、台中行约一周。本轮合并为“台中短行访友”节点。

## 新增地点与补强

- 新增 `place-taichung-agricultural-college`：按今日国立中兴大学校区近似定位；1958年台中农学院具体院舍位置待考。
- 补强 `place-tunghai-university`：加入1958年9月16日信中“午后赴东海大学看李华俊”的复访证据。

## 继续暂缓

- `010.有耳无膜.txt`：赴台大医院看刘凤翰病，确为到场，但属于病访短行，且地图已有更高权重的台大医院胡适节点，暂不另建。
- `012.十字街头筑隐居.txt`：陪家中老太太等赴木栅抽签，缺具体寺庙或签诗地点，暂不拆点。
- `024.不知情绪为何物.txt`：连日赴医院看病友，未给医院名，暂不入图。
- `046.蛋炒饭.txt`：台北云和街十号为胡蓉裳地址，不是李敖本人到场点。
- `049.好多个“最后一次”.txt`：台北市公共汽车站台大分站前写信、去治牙，属于微地点和日常短行，不入主图。
- `050.一封没写完的信.txt`：文星书店、敦煌书店为文化商业议论，不能证明本轮到场。

## 复查结论

《给马戈的五十封信》第一轮复查完成。当前本书入图两条主线：1955-1958年台大清谈派与马戈、1958年9月台中短行访友。其余线索保持候选，等待后续专题层或更强交叉证据。
