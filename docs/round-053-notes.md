# 第五十三轮：《李敖书序集续集》初筛

## 处理范围

- 来源：`《大李敖全集5.0》（wjm_tcy版）分章节/007.采访序跋类/003.李敖书序集续集/`
- 类型：采访序跋类第三本，包含书序、广告词、出版说明、题记和总序。
- 策略：延续书序类口径，只补强既有主线节点，不把出版品流通、历史对象空间或他人政治行动误作李敖本人足迹。

## 本轮结论

本轮不新增地图事件或地点。

《李敖书序集续集》的空间信息集中在出版流通、查禁、香港言论空间、报纸合订本、节目出版与政治人物传记。它能补证据，但不适合制造新的主图点位。

## 已补强证据链

- `event-1965-wenxing-banned`：`001.《李敖语录香港版》感言.txt` 记1965年5月27日至6月8日香港大会堂书展中，李敖与文星书店出版品占台湾参展册数近九成；随后国民党查禁其书、文星被封门。
- `event-1966-farewell-ten-books-censorship`：`002.《李敖告别文坛十书》广告词.txt` 提供前五册、后五册预约与出书时间，以及“台北邮箱24011号李敖”的通讯线索。
- `event-1991-qiushi-newspaper-end-review-magazine`：`003.《求是报合订本》广告词.txt` 说明《求是报合订本》问世，李敖编，只发售98部，并重申《求是报》是李敖个人主持的报纸。
- `event-1995-tv-xiaoaojianghu`：`018.《好命操作手册》序.txt` 记《李敖大全集》出版后，李敖在真相新闻网做《笑傲江湖》节目，1996年10月下旬起与施寄青、陈烨合作，每周解构算命迷信，三个月共十二集，1997年3月序于台北。

## 暂缓入图候选

- 1965年香港大会堂书展：是李敖/文星出版品登陆香港的强出版史线索，但文本未说李敖本人到场，暂不作为足迹。
- 下狱前夜“香江托孤文件”：可作为海外出版与禁书流通专题线索，但没有可定位地点。
- `010.《黄旗梦碎》序.txt` 所涉台北地方法院告发、立法院公听会、笑傲江湖节目安排，主要是钱达等人的行动链或电话安排，李敖本人到场不足。
- `013.《文茜半生缘》序.txt` 含陈文茜台中外家、美国、大陆、东势、海外经历，以及李敖与陈文茜两次吃饭、彭明敏回台后吃饭等线索，但均未给足地点或主线权重。
- `017.《调查局黑牢345天》序.txt` 补强李世杰调查局系列出版价值，但李敖出版社地址仍未由本书给出，暂不拆点。

## 下一轮复查提醒

- 复查时优先判断是否需要建立“香港言论/出版流通”专题层，把香港大会堂书展、香港版《李敖语录》、香江托孤文件串起来。
- 继续核对李敖出版社在1988年至1991年前后的办公地址，避免把金兰大厦、白宫大厦、《求是报》发行所和出版社空间混同。
- 《黄旗梦碎》《文茜半生缘》更接近政治人物/媒体人物专题，暂不进入人生主线。
