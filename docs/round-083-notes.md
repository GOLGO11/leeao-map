# 第八十三轮：《挑战李敖》初筛

处理日期：2026-05-09

本轮范围：`《大李敖全集5.0》（wjm_tcy版）分章节/010.节目演讲类/002.挑战李敖/`，共110篇节目文字稿，日期从1999年9月28日至2000年3月30日。

## 处理原则

《挑战李敖》处在2000年总统选举前后，地名密度极高，但大量是节目中评论的新闻地点、制度空间、历史地名或他人行动。本轮只纳入三类主线空间：

- 节目自身的媒体空间。
- 李敖本人到场的现场播出、现场录影、校园演讲。
- 与2000年总统选举直接相关且地点明确的大型公共演讲。

总统府、立法院、法院、饭店、电视系统业者、新闻局等在节目中被大量谈到，但除非文本能证明李敖本人到场，初筛不拆为主线足迹。

## 本轮新增入图

- `event-1999-2000-tv-challenge-leeao-global`：1999年9月20日至2000年3月30日，环球电视台《挑战李敖》。依据 `016.19991020挑战李敖.txt` “开播满一个月”和 `091.20000209挑战李敖.txt` 对吴子嘉请李敖进驻环球电视台的回顾。
- `place-global-tv-taipei`：环球电视台（台北，旧址待考）。文本没有门牌、楼层或摄影棚，因此只按台北市级坐标。
- `event-1999-2000-challenge-leeao-live-tour`：1999年10月21日至2000年1月10日，《挑战李敖》校园现场播出与全省讲透透。以一个聚合事件承接台北医学院、高雄中山大学、淡江大学、东吴大学、台湾大学、中正大学、政治大学、海洋大学、中华大学、成功大学等地点，避免事件过密。
- `event-2000-sun-yat-sen-memorial-hall-hometown-speech`：2000年1月21日，国父纪念馆“热爱乡土，共同打拼”演讲。依据 `080.20000120挑战李敖.txt` 的预告和 `081.20000121挑战李敖.txt` 的现场稿。
