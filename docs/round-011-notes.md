# 第十一轮笔记：《李敖风流自传》初筛

## 本轮结论

《李敖风流自传》大量重述李敖既有人生主线，本轮不重复制造密集事件点，而是按“重要事件 + 明确空间 + 李敖本人到场”原则处理。初筛后新增一个入图节点，并把多处既有主线节点的原文证据链补强。

## 新增入图

- `place-xindian-huayuan-xincheng-xiao-home`：新店花园新城（萧孟能家）。坐标按新店花园新城社区范围近似，非萧宅门牌，标记 `needs_review`。
- `event-1979-hu-yinmeng-first-meeting`：1979年9月15日，李敖在萧孟能花园新城家中初见胡茵梦。

## 既有地点修订

- `place-garrison-command-security-office`：由“警总保安处”细化为“警总保安处（博爱路）”，补入《李敖风流自传》“台北市博爱路警备总部保安处”“第五房”证据；坐标按博爱路警备总部范围暂定，仍需与六张犁看守所线索复查。
- `place-military-law-detention-center`：由无坐标“警总军法处看守所”细化为“景美军法看守所（警总军法处看守所）”，坐标按景美白色恐怖纪念园区旧址范围暂定。
- `place-military-law-room-8`：补入“景美军法看守所第八房”别名与说明；八号房属于内部空间，暂不单独设坐标，避免与景美看守所重复堆点。
