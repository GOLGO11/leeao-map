# 第九十一轮：《李敖大哥大》初筛记录

## 本轮处理范围

- 书目：`010.节目演讲类/006.李敖大哥大/`
- 文件：127篇节目文字稿，另有目录文件。
- 时间：2001年10月15日前后至2002年5月20日前后。
- 原则：只收李敖本人主持/连线的节目空间与明确到场点；节目中谈论的新闻地点、历史地点、候选人行程和来宾经历暂不入主图。

- 新增 `event-2001-2002-tv-leeao-dageda-cti`：中天电视《李敖大哥大》。起点据第1、2集张学良去世当天/次日线索推定到2001年10月15日前后；终点据第126集“2002年5月17号”和第127集“520”定到2002年5月20日。

- 新增 `event-2002-cti-cctv-mainland-dialogue`：2002年马年正月初一至初四《李敖对祖国发音》卫星连线专题。李敖在台北中天端，通过中央电视台与北京现场读者对话。

- `001.张学良是末代民族英雄.txt` 第13行：张学良当天去世，节目即作评论。
