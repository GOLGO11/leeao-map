# 第四十一轮：《李敖札记》初筛

## 本轮判断

- 《李敖札记》篇目多为政治短札、新闻札记与语录，不是连续日记或行旅记录。正文地名多数属于议论对象、新闻对象或他人行程，本轮不密集拆点。
- 主线价值最高的是出版查禁链：1986年《蒋介石研究》出版，1987年《蒋介石研究续集》漏网触发警总搜书压力。

## 本轮新增

- `event-1986-1987-chiang-research-publication-censorship`：1986年11月3日《蒋介石研究》出版，1987年2月19日警总因夜市见《蒋介石研究续集》漏网而搜查书摊。地点暂按 `place-taipei` 聚合，不拆夜市或书摊。

## 暂缓候选

- `024.动物园的感想.txt`：1984年9月10日李敖与导群午后逛动物园。地点明确但主线权重较低，且需复查当时是否为圆山动物园旧址，暂不入主图。
- `020.萧孟能窃占罪入狱.txt`、`110.掐死“伪”文星.txt`：萧孟能、台北地检处、台北监狱、土城看守所、台北市政府新闻处等线索适合后续“法律战/文星余波”专题。
- `057.匡复起点搬家了，重建基地动摇了.txt`、`091.工专长寿·台大短命.txt`：属于新闻对象或校史评论，不作为李敖本人足迹。
