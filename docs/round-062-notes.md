# 第六十二轮：《李敖书信集》复查

## 本轮定位

复查上一轮留下的护照/出境证链条、水晶大厦、2010年沪返台与蔡武到访、台北牢头、文物通信和马戈晚年照护线索。结论是：护照节点应保留并校正时间；其余候选目前仍不宜进入主线地图。

## 调整既有节点

- `event-1980-passport-exit-ban-battle`：复查后把结束日期由1980年11月15日改为1980年11月23日。11月15日是外交部领事事务处来函日期；李敖给朱抚松的第二封信署“李敖 1980年11月23日”，后记又说朱抚松收到这第二封信后终于给护照。故本节点时间范围改为1980年8月25日至11月23日。
- 本节点仍用 `place-taipei`。文本能证明锡安旅行社代办、外交部领事事务处来函、警备总部给出境证、买飞机票时出境证被扣留，但没有给出旅行社、领事事务处、售票处或机场的稳定旧址，也不能证明相关动作都发生在同一可落点空间。

## 复查后仍不入图

- 水晶大厦：`063.1981年李敖家书.txt` 只证明李敖一度从金兰搬到水晶、后来住回金兰；未给门牌与坐标。本轮继续把它作为金兰大厦住处节点中的说明，不另建地点。
- `068.2010年致蔡武信残本.txt`：2010年9月5日信中写“五天前自沪返台”，刘长乐送他上飞机，并称蔡武部长到“我家”拜访。线索有晚年两岸文化交流价值，但没有说明上海活动地点、机场名或台北“我家”门牌；2010 年他是否已住回金兰，也需与其他材料交叉确认，所以暂不新增沪台路线或家访节点。
- `044.给台北牢头的一封信.txt`：主要是萧孟能服刑、荣总医院证明和病监待遇问题。它触及台北监所和荣总，但不是李敖本人新增足迹。
- `067.2002年4月2日致杨仁恺信.txt`：台北故宫、台北历史博物馆等是文物对象空间，不是李敖到访空间。
- `069.2015年致王尚勤短信.txt`：附录中桃园机场、徐州路台大校友会馆、荣总、振兴医院等均是陈彦增照护马戈的路径，不是李敖本人足迹。

## 复查结论

本轮不新增事件、地点或事件-地点关系。`event-1980-passport-exit-ban-battle` 保留为本书主线成果；下一步若要拆细，应优先查外交部领事事务处1980年前后办公地、锡安旅行社旧址、警备总部保安处出境证办理场景，以及扣留出境证发生在售票处还是机场。
