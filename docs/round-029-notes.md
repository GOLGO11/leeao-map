# 第二十九轮：《大学札记》初筛

## 处理范围

- 来源：`006.沉思日记类/001.大学札记/`
- 时间范围：1957年4月至1957年11月
- 本轮策略：日记类文本只收入明确外出行程、可定位学术空间、能补强既有主线地点的材料；私人会面地点不凭人名硬推门牌。

## 新增入图

1. `event-1957-nangang-academia-sinica-study`
   - 1957-04-28，李敖与姚从吾、庄申在温州街口会合，同车赴南港中央研究院读书，并到历史语言研究所书库谈学问。
   - 新增地点：`place-academia-sinica-nangang`。
   - 坐标按中央研究院南港院区校级近似，具体史语所书库位置待考。

2. `event-1957-tunghai-beigou-antiquities`
   - 1957-08-14，李敖与台大历史系同学参观东海大学，午后赴北沟参观古物。
   - 1957-08-23，再赴北沟看第三期古物展览。
   - 新增地点：`place-tunghai-university`、`place-beigou-palace-museum-wufeng`。
   - 北沟坐标按台中雾峰北沟一带近似，旧库房/展览空间需要复查。

## 暂缓候选

- 殷海光家：`022.见殷海光.txt`、`116.见殷海光.txt` 两次会面重要，但无地址。
- 台静农家、方豪处、吴相湘会面、姚从吾谈话、陈淑平家：均缺可靠地点，暂留师友网络候选。
- 文星书店：`117.在文星书店的一个感想.txt` 可作为早期文星接触线索，但没有店址，不能直接套用衡阳街或峨眉街节点。
- 思恩堂、慈航禅院：有地名但当前主要呈现心境与校园生活，是否建点待复查。

## 复查重点

- 核查北沟故宫文物旧址、东海大学1957年可参观点、中央研究院史语所书库位置。
- 查殷海光、台静农、方豪等私人会面地点是否能由其他书互证。
- 查1957年文星书店实际店址，决定是否将该条并入既有文星书店地点。
