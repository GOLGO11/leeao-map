# 第六轮笔记：《李敖回忆录》复查

## 本轮结论

《李敖回忆录》第一轮复查聚焦初筛暂缓点。最值得补入地图的是杭州南路文献会：它不是普通背景地名，而是李敖从新店山居回到台北、进入陶希圣主持的“中华民国开国五十年文献编纂委员会”的工作与居住节点。

## 新增入图

- `event-1962-wenxianhui-hangzhou`：1962年2月1日至1963年5月前后，入文献会并住杭州南路办公处。
- `place-taipei-wenxianhui-renai-hangzhou`：开国五十年文献会（仁爱路二段9巷27号）。

## 证据互证

- 《李敖回忆录》写“我就从新店迁回台北，住进杭州南路文献会”。
- 《李敖自传与回忆》保存吴相湘信，写“仁爱路2段9巷27号该文献会”。
- 同篇又引蒋君章回忆，称办公室在“仁爱路与杭州南路之交”。

这三处证据足以把文献会从候选地点提升为主线事件点；但坐标仍为旧门牌近似，保留 `needs_review`。

## 继续暂缓

- 胡须张、福华大饭店、太平洋崇光百货：它们与东吴任教促成过程有关，但不必拆成主线点。
- 真相新闻网、TVBS、录影棚：文本能证明电视阶段，但没有可靠录影地点，本轮继续维持台北市级点。
- 国泰医院：已经入图，但院区与旧称仍需进一步考证。
