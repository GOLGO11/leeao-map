# 第四轮笔记：《我最难忘的事和人》初筛

## 本轮新增重点

- 温州街73号台大第一宿舍第四室：补足台大历史系后期住居。
- 信义路/安和路国泰信义公寓：郑南榕1968年首次见李敖的地点，并与《文星》资料室相连。
- 日月潭与九族文化村：1989年《潭畔寻思录》的现场。
- 新竹交通大学：1989年12月21日《交通快与慢》演讲。
- 中兴大学中兴湖：1990年6月以湖为题的空间记忆。
- 台湾开明书店中山北路一段77号：大学时期常去、1991年前后重访的书店旧址。

## 本轮入图事件

- 台大历史系后期：住温州街73号台大第一宿舍第四室。
- 1968年暑期：郑南榕第一次到李敖信义路/安和路寓所。
- 1989年7月26日：日月潭畔写《潭畔寻思录》。
- 1989年12月21日：新竹交通大学演讲《交通快与慢》。
- 1990年6月：写中兴大学中兴湖。
- 1991年前后，1992年2月9日追记：重访台湾开明书店。

## 暂缓入图说明

本书后半部大量内容是人物纪念、传记材料和政治评论，其中出现许多他人足迹。当前地图不把这些地点直接并入李敖路线，除非原文明确显示李敖本人到场、通信、演讲、居住或在某处形成直接空间记忆。
