# 第六十一轮：《李敖书信集》初筛

## 本轮定位

《李敖书信集》进入书信函件类第二本。自序说明原书收1966年4月25日至1986年3月5日二十年间六十封信，当前电子版又附录若干后搜集书信。书中地点很多，但多数属于收信人地址、论战对象、报刊材料或政治议论中的地名；初筛只收能够改写李敖人生空间主线的事件。

## 新增入图

- `event-1980-passport-exit-ban-battle`：1980年8月至11月，为护照而战与出境受阻。`006.为“护照”而战.txt` 记锡安旅行社8月25日替李敖向外交部送件申请护照，外交部以婚姻状况“经查与事实不符”为由要求更正，11月15日由外交部领事事务处来函；后记又说朱抚松收到第二封信后终于发给护照，但有关方面仍不准出境。`009.我才不走呢！.txt` 又回顾1980年警备总部给出境证、外交部给护照，但买飞机票时出境证被扣留。由于外交部领事事务处、锡安旅行社与机场环节的具体旧址尚未复核，本轮以 `place-taipei` 市级锚点入图。

## 补强既有节点

- `event-1978-1998-dunhua-south-road-residence`：`063.1981年李敖家书.txt` 第27、29、31行补出金兰大厦/水晶大厦之间的住居切换。文本能明确补强金兰大厦长期住处线索；水晶大厦目前无可靠门牌与坐标，不另建点。
- `event-1981-second-imprisonment`：同篇第23、37、39行写家人“到牢里看我”、李敖“牢中独坐”并“今天我在牢里写这封信”，可补强1981年第二次入狱期间的牢中书信证据。

## 暂缓入图候选

- `044.给台北牢头的一封信.txt`：1984年9月12日致庆璧典狱长，涉及萧孟能、荣总病历、台北监所病监待遇等，但主要是他人服刑与监所管理问题，不是李敖本人新增足迹。
- `061.1958年9月20日致胡适信.txt`：署“台湾”，缺少具体地点，不另拆台大或台中点。
- `068.2010年致蔡武信残本.txt`：提到“五天前自沪返台”、凤凰电视刘长乐送机、文化部蔡部长到李敖家拜访、北京三联拒印《阳痿美国》，具有晚年两岸文化交流价值；但“我家”门牌未明，且赴沪行程需结合其他材料复核，暂留候选。
- `067.2002年4月2日致杨仁恺信.txt`：涉及台北故宫、台北历史博物馆、辽宁/沈阳博物馆书画真伪争议，主要是文物对象空间，不是李敖到访路线。
