# 第七十五轮：《李敖书启集》初筛

处理时间：2026-05-09

来源范围：`《大李敖全集5.0》（wjm_tcy版）分章节/008.书信函件类/009.李敖书启集/`

## 本轮原则

本书覆盖1991年9月至1999年1月，材料以书启、公开信、邻居争议、司法函件、报刊更正和文化人物通信为主。初筛不把每封公开信对应机关都拆成地点，只处理能补强李敖长期生活空间、教学空间、司法/出版主线，或能新增晚年空间轨迹的材料。

## 新增入图

- `event-1995-caoshan-yangmingshan-residence-plan`：据 `043.我与邓丽君后一章.txt`，李敖1995年5月24日致林永丰信中说明自己已在草山（阳明山）订下山居新建楼房，并说“将来计划死在那里，台湾是我埋骨之地”。本轮复用 `place-yangmingshan-qingshan-jianwo-lou`，把阳明山书房/山居线索提前到1995年。

## 已补强证据链

- `event-1978-1998-dunhua-south-road-residence`：补入 `003.怨结金兰.txt` 和 `026.政骚扰.txt`。前者显示1991年底李敖仍以金兰大厦主任委员身份处理CD座邻居争议，并自称搬入金兰十四年；后者给出“台北市敦化南路1段306号金兰大厦李敖收”作为1993年证明文件寄送地址。
- `place-taipei-dunhua-south-road-residence`：新增“敦化南路1段306号金兰大厦”别名，并记录金兰门牌存在496号与306号两种写法，后续需复查旧制/新制门牌或路段写法。
- `event-1993-soochow-teaching`：补入 `030.给台大校长的公开信.txt`，该信以台大哲学系事件为背景，对比章孝慈请李敖到东吴任教，强化东吴教学节点的同期自述。
- `event-1987-taichung-book-raid-lawsuit`：补入 `028.致《民众日报》李瑞标.txt`，该文确认1993年2月22日最高法院判决台中市政府败诉定谳，并由张桂贞委托李敖代表洽接赔偿。

## 暂缓入图候选

- `039.为法官黄小莹执法偏颇质国民党司法院长施启扬函.txt`：新店市华城一路20号强制执行案有明确地址，也写到李敖到场指认物件，但核心是王志慧房产与段宏俊/钟惠璇纠纷，暂列法律专题候选，不压入主线地图。
- `017.黄大洲听着！.txt`、`019.致黄大洲.txt`：东丰街56号违建检举属于市政/邻里公共安全案件，未形成李敖主线空间轨迹，暂缓。
- `041.我与邓丽君.txt` 与 `043.我与邓丽君后一章.txt`：邓丽君灵堂是媒体误报，李敖明确否认到场；不建立灵堂地点，只采后一文中的草山山居线索。
- `046.致法务部调查局.txt`：秦孝仪《孙中山墨迹》鉴定函涉及法务部调查局与台湾高等法院，但是诉讼材料流转，不是李敖到场点，暂不入主图。
