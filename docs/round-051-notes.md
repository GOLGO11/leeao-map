# 第五十一轮：《李敖书序集》初筛

## 处理范围

- 来源：`《大李敖全集5.0》（wjm_tcy版）分章节/007.采访序跋类/002.李敖书序集/`
- 类型：采访序跋类第二本，主要由书序、推荐文、题记和新版缘起组成。
- 策略：只收入李敖本人明确到场、能补强人生主线或出版主线的空间证据；被介绍书籍里的历史地点、他人经历地点和政治案件对象空间暂不入主图。

## 本轮结论

本轮不新增地图事件或地点。

《李敖书序集》里地点不少，但大多是李敖借书序处理政治史、出版史和人物史。例如《汪政权的开场与收场》《紫禁城的黄昏》《满宫残照记》《侍卫长谈毛泽东》等篇的北京、南京、重庆、北平、上海等地名，主要属于书中对象的历史舞台，不是李敖足迹。

## 已补强证据链

- `event-1972-military-law-detention`：`007.《调查局研究》序.txt` 记1971年2月28日李敖自警总保安处移到景美军法看守所，先住第二房，至1975年12月22日解送土城仁爱教育实验所。
- `event-1972-1975-military-law-room-8`：同篇记1975年9月22日改判后，李敖已搬到第八房，李世杰在第九房。
- `event-1975-tucheng-renai`：同篇记1975年12月22日李敖被通知收拾行李，解送土城仁爱教育实验所，并接到1976年11月19日出狱。

## 暂缓入图候选

- 1986年6月30日，李敖亲赴李世杰家，约午饭并说动他展开《调查局研究》写作计划；地点只写“他家”，不能可靠收窄。
- 1988年6月6日，唐德刚从美国来，由陈宏正陪同来到李敖家，签约由李敖出版社重排《李宗仁回忆录》；可作为出版史候选，但本轮不以住处推定新增主图点。
- 1966年8月31日，李敖在台北牯岭街书摊再见《海国男儿》并买下；适合后续“书摊/禁书流通”专题层，不进入人生主线地图。
- 1990年4月5日，李敖亲访罗开嘉并转交宋希濂原信；地点未明。
- 1991年11月12日《历年办理匪案汇编》序涉及台大法学院支部、保安司令部、桃园监狱、延平南路看守所、景美/新店等白色恐怖材料；价值高，但多为案件对象空间，不是李敖本人足迹。
