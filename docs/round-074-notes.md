# 第七十四轮：《李敖书函集》第一轮复查

处理时间：2026-05-08

来源范围：`《大李敖全集5.0》（wjm_tcy版）分章节/008.书信函件类/008.李敖书函集/`

## 本轮结论

本轮在第七十三轮基础上复查公开信、法律抗辩、报刊通信和候选地点。复查后新增一个主线节点，校正一个既有住处节点，并对一个书展节点下调确定性。原则仍是只收入李敖生命轨迹或出版/司法现场有明确锚点的事件；单纯收信机关、对象人物所在地、读者来信地点不单独入图。

## 新增入图

- `event-1987-sun-yat-sen-research-post-martial-law-censorship`：据 `002.查禁《孙中山研究》的荒谬理由.txt`、`003.给新闻局局长邵玉铭的公开信.txt`、`010.请检察官辞职.txt`，补入1987年9月25日至11月24日解严后《孙中山研究》遭台北市政府新闻处查禁、同时《李敖千秋评论丛书》第七十一期遭处分，并由行政查禁延伸到台北地检处移送的节点。因本书未给出可靠机关门牌，地点按 `place-taipei` 台北市级聚合。

## 已校正或补强

- `event-1978-1998-dunhua-south-road-residence` 保持既有 id 不变，但时间标签由“约1978年至1998年前后”改为“约1977年至1998年前后”。依据 `018.给邻居的一封信.txt`，李敖自述“自从1977年搬入金兰大厦、成为各位邻居后，十一年来”，并记1989年被推为B座管理委员和主任委员。
- `place-taipei-dunhua-south-road-residence` 的说明补入金兰大厦1977年起居住与1989年管委会线索，同时继续提醒白宫大厦、林肯大厦、金兰大厦不可混同。
- `event-1990-national-book-exhibition-sun-yat-sen-censorship` 保留国父纪念馆锚点，但确定性由 `high` 下调为 `medium`。理由：`032.书而不展，还叫什么书展？.txt` 明列国父纪念馆为协办单位和收文者，但原文只写“首在台北展出”，未直接写展场具体空间。后续若找到展场公告或书展资料，可再升格。

## 继续暂缓

- `017.《乌鸦评论》及其他.txt`：复查确认1989年4月14日演讲存在，且现场站立两小时、场地过小，但仍没有会场名称；“下次能租下国父纪念馆类似的场所”明确说明本次不是国父纪念馆，继续不入图。
- `031.中央研究院史语所岂可掩护李济、许倬云！.txt`：仍是函购与公开质问，不能证明李敖到南港史语所，不补南港点。
- `040.打官司的讨论.txt`、`043.打官司的再讨论.txt`：可补强《求是报》办报责任、档案仓库和匿名作者争议；“苦茶庵主”桃园中正机场明信片是他人行动，不作为李敖地点。
- `048.宋英、黄信介等岂可违反傅正遗愿！.txt`：南港自由墓园是傅正/雷震等人的身后事讨论空间，不是李敖到场轨迹，暂留人物专题候选。
- 王剑芬居留案、新闻评议会/新闻局多次公开信、中央日报/自由时报辩驳书争议等，均有制度史价值，但本轮不拆为主线地图点。
