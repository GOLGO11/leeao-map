# 第三轮笔记：《李敖自传与回忆续集》初筛

## 本轮新增重点

- 台中存德巷一号：1952年6月15日，徐武军带李敖拜见钱穆。
- 台中存德巷十三号台中一中宿舍：李敖回忆当时住处与钱穆住处相近。
- 台中车站：1952年10月1日，李敖递长信给胡适。
- 文星书店衡阳街口小摊：文星书店早期经营地点。
- 文星书店峨眉街店：文星事件中被提及的现代化书店空间。
- 信义路国泰信义公寓：1970年至1971年长期跟监、软禁语境下的关键居所。
- 警总保安处六张犁看守所：1971年被捕后的关押地点，按原文“和平东路底与卧龙街交叉口、六张犁公墓山底下”暂定坐标。
- 台北看守所（土城立德路二号）：第二次政治犯时期的具体监所地点。

## 本轮入图事件

- 1952年6月15日：在台中存德巷拜见钱穆。
- 1952年10月1日：在台中车站递信给胡适。
- 1961年前后：李敖进入《文星》视野。
- 1965年12月：《文星》杂志被封。
- 1968年3月至4月：文星书店宣布结束。
- 1970年1月至1971年3月19日：在信义路寓所遭长期跟监。
- 1971年3月19日至1972年2月27日：被捕后关押于警总保安处六张犁看守所。
- 1981年8月10日至1982年2月10日：第二次入狱，地点精确到土城台北看守所。

## 待复核地点

- 台中存德巷一号、十三号旧址坐标。
- 文星书店衡阳街口小摊的具体街口。
- 文星书店峨眉街店门牌与营业时期。
- 文星杂志编辑部或发行办公地点；目前不能简单等同于书店。
- 警总保安处六张犁看守所旧址坐标。
- 台北看守所土城立德路二号坐标与1981年时的实际关押区域。

## 暂缓入图说明

《续集》中还出现温州街台大第一宿舍、台大医院、台大文学院门口、中央研究院等地点。它们有文本证据，但本轮先不全部入图，理由是当前地图保持“重要事件”密度；这些地点更适合在后续复核胡适关系、台大时期或住居变迁专题时再决定是否提升为主线事件。

## 复查要求

本轮只算《续集》的初筛。下一轮复查需要逐篇回到原文，检查每条已入图事件是否仍应保留、坐标精度是否需要下调或上调、文星书店与文星杂志地点是否被错误合并，以及是否有被初筛漏掉的关键足迹。
