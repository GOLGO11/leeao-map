# 第四十三轮：《李敖五五日记》初筛

## 本轮判断

- 本书是1990年4月25日至1991年4月10日的连续日记，地名密度很高，但许多是饭局、电话、探病、短程购物或日常会议。本轮只收能支撑主线的公开活动、出版节点与可校正既有地点的证据。
- 这本书最重要的空间价值，是把1990年前后的校园演讲、电视露面、台中行与《求是报》筹办放回日记日期中。

## 本轮新增

- `event-1990-feng-chia-lecture`：1990年4月27日南下台中，在逢甲大学演讲《逢甲甲与乙》。新增 `place-feng-chia-university`，坐标按校级近似，具体馆舍待考。
- `event-1991-qiushi-newspaper-launch`：1991年2月11日《求是报》试版，2月28日出刊。地点暂按 `place-taipei`，下一轮复查林肯会议、公司与编辑地点。

## 已补强

- `event-1990-nchu-zhongxing-lake`：由原先的“地点对象型空间记忆”改为有实际到场证据的节点。日记明确写1990年5月11日清晨在中兴大学快走并照相一个半小时。
- `event-1990-ctv-writer-directory-appearance`：补入1990年6月9日夜在中视录影四小时、6月12日曾谁芬来电“报备”删减发言情况等日记证据。

## 暂缓候选

- 1990年5月10日“中台病不病”演讲：文本有学生接站、演讲和夜宿台中，但“中台”机构定位需复查，暂不建点。
- 1990年11月至12月全美华人协会波士顿成就奖：李敖获奖并以录音代替现场发言，但文本明示未赴美，不作为本人足迹入主图。
- 1990年7月、8月台中开庭：法律史价值明确，但案件、法院与既有查禁线索需要交叉后再处理。
- 林肯会议、公司、《求是报》编辑地点：高频出现但尚无可靠地址，不伪精确。
