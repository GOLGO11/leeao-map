# 第七十九轮：《给马戈的五十封信》初筛

处理时间：2026-05-09

来源范围：`《大李敖全集5.0》（wjm_tcy版）分章节/008.书信函件类/011.给马戈的五十封信/`

## 本轮原则

本书目录标示时间范围为1958年8月17日至1958年12月10日，主体是李敖写给台大朋友马戈的五十封旧信。跋文写于1966年8月26日，回看1955年至1958年台大历史系时期的交游。初筛不把信中零散的约会、电影、朋友家、邮局、公车站都拆成地图点；只处理能够代表李敖早年思想交往和台大空间记忆的节点。

## 新增入图

- `event-1955-1958-ntu-margo-qingtan`：据 `051.跋《给马戈的五十封信》.txt`，李敖1955年暑假后重考入台大文学院历史系，在历史系迎新会上结识马戈；此后三年，二人和朋友常在图书馆、小饭店、弹子房、电影院、台大校园草地等处清谈。1958年暑假马戈毕业南下受预备军官训练，李敖遂在三个多月里大写书信。本轮以 `place-ntu` 入图，作为台大清谈派与青年思想交往节点。

## 已补强证据链

- `event-1955-ntu-history`：补入本书跋文，强化“自动退出法律系、重考入历史系”与历史系迎新会的空间背景。
- `place-ntu`：备注补入台大历史系迎新会、台大校园草地清谈线索。

- `012.十字街头筑隐居.txt`：陪家中老太太等赴木栅抽签，缺少具体寺庙/地点，暂不拆点。
