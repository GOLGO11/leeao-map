# 第七十三轮：《李敖书函集》初筛

处理时间：2026-05-08

来源范围：`《大李敖全集5.0》（wjm_tcy版）分章节/008.书信函件类/008.李敖书函集/`

## 本轮原则

本书覆盖1987年9月至1991年7月，公开信和对象空间非常多。初筛只收三类：能补出李敖居住/编辑基点的空间，明确发生在公共场所的出版审查现场，以及具有法院锚点的法律事件。新闻局、新闻评议会、史语所等收信机关，如只构成通信对象而非李敖足迹或明确现场，本轮暂不入主图。

## 新增入图

- `event-1987-1988-white-house-building-residence-management`：据 `011.从小事看认真.txt`，李敖到白宫大厦做邻居五个多月后，被选为十二楼代表和主任委员；文中又自述“住在白宫大厦邻厦——金兰大厦十年”。因信内出现“1988年2月起”和第二信“1988年2月14日”，第一信署“1986年2月13日”疑为录入/沿用错误，本轮按1987年下半年至1988年2月处理。地点复用并扩展 `place-qiushi-newspaper-dunhua-490`，改名为“白宫大厦 / 求是报社（敦化南路490号）”。
- `event-1990-national-book-exhibition-sun-yat-sen-censorship`：据 `032.书而不展，还叫什么书展？.txt`，1990年12月24日李敖代李敖出版社致函行政院新闻局、国父纪念馆与台湾商务印书馆，指“中华民国七十九年度全国图书展览”在国父纪念馆展出时，《蒋宋美龄通奸》《共产党李登辉》《施启扬要红包》三书已列清单和目录，却被工作人员暗中从书架取下。地点为既有 `place-sun-yat-sen-memorial-hall-taipei`。
- `event-1991-hu-qiuyuan-libel-verdict`：据 `041.胡秋原诽谤李敖，被判处拘役四十天.txt`，李敖自诉胡秋原诽谤案，台北地方法院七十九年度自字第986号判决胡秋原散布文字诽谤，处拘役四十日。地点复用 `place-taipei-district-court`，仍按院本部/司法机关区域近似。

## 已补强证据链

- `event-1987-taichung-book-raid-lawsuit`：补入 `020.斥台中一中校长陈继统.txt`，作为张桂贞台中一中宿舍搬迁/法律战的后续证据。该文说明台中一中委托律师发函，李敖代收并通知母亲赶回台湾法律解决，且明确张桂贞自1949年住进台中一中宿舍达四十年。
- `event-1991-qiushi-newspaper-launch`：补入 `037.答辅大女学生.txt`，作为《求是报》读者来信和编辑内容生态的本书证据。
- `place-qiushi-newspaper-dunhua-490`：由单纯“求是报社”地点扩展为白宫大厦综合楼宇；保留《求是报》发行所/编辑业务地址，同时登记1987年前后李敖居住和管委会事务线索。

## 暂缓候选

- `017.《乌鸦评论》及其他.txt`：可证明《乌鸦评论》办二十四期后告停，也可证明1989年4月14日李敖演讲存在、现场站了两个多小时，但文本没有给出会场名称；“下次可租国父纪念馆类似场所”不能反推本次会场，暂不建点。
- `031.中央研究院史语所岂可掩护李济、许倬云！.txt`：主要是函购史语所出版品与公开质问，不能证明李敖到南港中研院；暂不补南港节点。
- `040.打官司的讨论.txt`：对《求是报》报社责任、匿名作者和机场明信片有材料价值，但主要发生在报社通信与档案查找中，已由《求是报》节点承接，不另拆机场或法院点。
- `044.讨论《北京法源寺》.txt`：证明《北京法源寺》单行本封面设计与出版准备，但缺少可入图的独立地点。

## 下一轮复查重点

- 复核白宫大厦、金兰大厦、林肯大厦三个敦化南路空间的关系，避免把住处、报社、出版社和会议地点混同。
- 尝试查找1989年4月14日演讲会场；若能证实地点，再考虑加入演讲节点。
- 复核全国图书展当年台北展区是否确在国父纪念馆主体空间，及是否还有高雄、台中巡展后续材料。
