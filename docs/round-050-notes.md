# 第五十轮：《李敖报刊集》复查

处理日期：2026-05-08

## 本轮目标

- 对第四十九轮《李敖报刊集》初筛结果做第一轮复查。
- 重点检查《千秋评论》节点是否应从“百期”延展至“十年停刊”，以及四季出版公司、小书书报社等出版执行空间能否可靠入图。

## 复查补强

### 《千秋评论》节点延展至十年停刊

- 更新事件：`event-1981-1989-qianqiu-review-prison-to-hundred-issues`
- 更新内容：日期终点由1989年底出满百期延展到1991年9月26日停刊告白。
- 新增来源：`009.《千秋评论》停刊告白.txt`
- 证据要点：
  - 第一期开于1981年9月。
  - 1991年9月恰满十周年。
  - 总计一百二十期、一百二十一册。
  - 因《求是评论》（《李敖求是评论杂志》）即将创刊，为集中主力，决定把办了十年的《千秋评论》停刊。

## 继续不拆点

- 四季出版公司、林秉钦、叶圣康：文本能证明预编六册由林秉钦转叶圣康的四季出版公司出版，但没有可靠地址；不另建地点。
- 小书书报社、苏荣泉：`011.《乌鸦评论》休刊告白.txt` 只说明订户退款和折购由小书书报社苏荣泉个别通知，未给地点；不入图。
- 李敖出版社、文星书店丛书：文本只说明同一时期每月出版链的负荷，不是明确到场地点；不入图。
- 《万岁评论》与《乌鸦评论》：复查确认它们是《千秋评论》出版链的内部阶段。《万岁评论》后并入《千秋评论》，《乌鸦评论》五个半月后休刊，当前不单独增加主图点位。

## 复查结论

- 《李敖报刊集》复查后不新增地点，保留一个聚合事件。
- 事件标题和时间线改为“《李敖千秋评论》从牢中创刊至十年停刊”，更符合本书证据。
- 主图仍以土城看守所作为该出版链的空间锚点；出版公司和发行空间待未来有可靠地址后再拆专题层。
