# 第九轮笔记：《李敖议坛哀思录》初筛

## 本轮结论

《李敖议坛哀思录》不是普通回忆录，而是立法院时期的政治回忆、问政记录与论战材料。初筛只把能改变主线地图理解的空间节点入图，大量公报和发言文本先留给复查轮。

## 已入图新增

- `place-sun-yat-sen-memorial-hall-taipei`：台北国父纪念馆。
- `event-2004-sys-memorial-campaign-speech`：2004年11月14日，李敖在国父纪念馆作唯一正式竞选公开演讲。
- `event-2006-legislative-yuan-tear-gas-arms-purchase`：2006年10月24日，立法院程序委员会喷瓦斯阻军购案。
- `event-2006-taipei-mayor-campaign`：2006年10月至12月，参选台北市长传播政治理念。
- `place-yangmingshan-qingshan-jianwo-lou`：阳明山/草山“青山见我楼”书房，门牌未明，按区域暂定。
- `event-2006-2007-qingshan-jianwo-lou-yitan-writing`：2006年12月至2007年9月，在“青山见我楼”书写、署记《李敖议坛哀思录》相关文字。

## 已修正既有事件

- `event-2004-legislator`：把日期从笼统的2004年校准为2004年12月11日开票日，并补入本书对立法院时期“反对”“嘲弄”和反军购主轴的证据。

## 取舍说明

- 立法院问政、公报、军购质询非常密集，初筛不逐条拆点；否则台北地图会被同一地点的重复事件淹没。
- 市长选举先用台北市级地点承载，因为本轮文本明确的是参选与政见传播，未给出足够可靠的具体场馆。
- “青山见我楼”文本证据充足，但缺门牌；因此保留 `needs_review: true` 和 `coordinate_precision: area_approx`。
