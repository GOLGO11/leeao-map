# 第四十九轮：《李敖报刊集》初筛

处理日期：2026-05-08

## 本轮目标

- 进入采访序跋类第一本《李敖报刊集》。
- 本书以报刊创刊词、停刊告白、杂志导言和出版回看为主，重点不是行旅地点，而是媒体出版节点。
- 继续控制主图密度，只新增对李敖人生轨迹有主干意义的媒体节点。

## 本轮新增

### 《李敖千秋评论》从牢中创刊并出满百期

- 事件：`event-1981-1989-qianqiu-review-prison-to-hundred-issues`
- 地点：`place-taipei-detention-center-tucheng`
- 时间：1981年9月1日至1989年底
- 来源：`005.被封杀的《千秋评论》.txt`、`006.《万岁评论》缘起.txt`、`008.《千秋评论》一百期期期期期期.txt`
- 证据要点：
  - 《千秋评论》原拟登记为杂志，1981年4月18日取得执照后，一个多月后又被官方以《出版法》第11条“技术击倒”。
  - 李敖改以《李敖千秋评论丛书》形式出版；1981年8月10日第二次入狱当天清早，他将预编六册交给林秉钦转四季出版公司。
  - 1981年9月1日起，《李敖千秋评论丛书》在他人在牢中时“越狱”而出；1989年底出足一百期。
- 坐标处理：锚定土城看守所，因为“第一期在黑牢里创刊”是本地材料最清晰、也最具空间意义的起点。四季出版公司、发行与后续制作空间未在本轮可靠收窄，不另建点。

## 已补强证据链

- 《文星》：`002.《文星杂志选集》导言.txt` 补强 `event-1961-wenxing-li-ao-appears` 与 `event-1965-wenxing-banned`，明确1957年11月创刊、1965年12月被封、总共九十八期，以及第49期起李敖出现使《文星》改观。
- 《求是报》创刊：`012.《求是报》1991年2月27日创刊.txt` 补强 `event-1991-qiushi-newspaper-launch`，新增东方版/西方版同步发行与1991年1月16日局版台报字第0354号执照线索。
- 《求是报》停刊与《李敖求是评论》：`017.《求是报》停刊告白.txt` 与 `018.《李敖求是评论杂志》发刊词.txt` 补强 `event-1991-qiushi-newspaper-end-review-magazine`，明确《求是报》1991年8月20日停刊、共172号，以及《千秋评论》和《求是报》双双完成阶段使命后转办《李敖求是评论杂志》。

## 暂缓入图

- 《万岁评论》与《乌鸦评论》：确认它们属于1980年代李敖每月出版链，但当前不拆为单独地图点，先并入《千秋评论》媒体链。
- 《求是报》美国西方版：有加州登录号和施复华主持业务，但不是李敖本人足迹，先作为发行网络候选，不入主图。
- 民众日报高雄演讲提议、新闻对象地名、历史人物地名等，均非本人到场足迹，不入图。
