# 第五十七轮：《李敖访谈录1990-2018》初筛

## 本轮定位

《李敖访谈录1990-2018》收 1990 年至 2018 年媒体专访、专题报道与身后回顾，时间跨度覆盖李敖从诉讼、电视、总统参选、立委、神州文化之旅到晚年病中访谈的后半生公开形象。初筛策略是：只收访谈文本中明确出现李敖实际到场、并能改变主线空间理解的地点；媒体所在地、谈论对象地名、历史比喻和泛泛电话采访均不入主图。

## 新增入图

- `event-1990-taipei-district-court-wu-dayou-case`：1990年11月21日，台北地方法庭吴大猷、王世杰子女诽谤案应讯。`001.19901120戴小华专访李敖.txt` 记戴小华11月20日晚访李敖寓所，次日上午随李敖前往台北地方法庭；法官因被告缺席择日再审。此节点复用 `place-taipei-district-court`，补出1990年前后李敖日常法律战的实际到场。
- `event-1999-new-party-presidential-campaign-ticc`：1999年8月18日，台北国际会议中心新党六周年党庆参选演讲。`008.19990819《中国时报》专访李敖.txt` 记李敖可望代表新党竞选总统，前一日晚间出席新党六周年党庆并发表专题演讲，近五千名群众涌入台北国际会议中心。
- `place-taipei-international-convention-center`：台北国际会议中心。当前按现址近似并标记 `needs_review`，复查轮需核该次党庆具体厅别与活动日期。

## 补强既有线索

- 2005 年神州文化之旅：`039.20050702《美国侨报》专访李敖.txt` 与 `040.20050919《中国时报》专访李敖.txt` 提供行前情绪、凤凰卫视安排、北大资料赠送、基隆至香港再转北京的替代路线构想等背景；`042.20050929《南方周末》专访李敖.txt` 则补出北大校办公楼二层演讲报告厅、清华演讲、复旦题词等报道细节。本轮暂不改动既有 `event-2005-mainland-lectures`，留到复查轮统一补证。

## 暂不入图

- `006.19990403《长江日报》专访李敖.txt` 为拨通李敖台湾家中电话采访，不能定位具体住处。
- `007.1999年6月PLAYBOY专访李敖.txt` 写“访问了李敖和他的书房”，但未给门牌；可与后续东丰街/阳明山书房互证，暂不落点。
- `027.20031215《TVBS》李敖专题.txt` 明写李敖摄护腺癌手术后回阳明山住处静养，并提到东丰街书房可能拍卖；但“阳明山住处”与既有“青山见我楼”是否同一空间、东丰街书房门牌均待复查。
- `073.20170222《亚洲周刊》专访李敖.txt` 记2017年2月22日晚来到李敖台北的家，并称阳明山有专门书房；`075.20170411《中国报》报导.txt` 又称李敖从阳明山搬下来住台北昂贵地段。这些是晚年住处重要候选，但未给地址，暂不新增主图点。
