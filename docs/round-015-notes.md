# Round 015 Notes - 《传统下的独白》第一轮

## 类别切换

本轮开始进入 `002.精品散文类`。与自传回忆类不同，精品散文中的大量地名是历史例证、文化论述对象、报刊出版地或外部引用，不一定代表李敖本人到场。因此本类初筛采用更严格的入图原则：

- 李敖本人行动、居住、写作署记、明确发表/出版事件，可以入图或补强既有事件。
- 历史人物、制度批评、文学例证中的地名，先不入图。
- 报刊、出版社、广播机构只在形成李敖公共轨迹节点时入图；一般出处只做证据。

## 本轮新增入图

- 1963年9月25日，《传统下的独白》出版。再版自序称出版后两个星期即有再版机会，并提到中国广播公司介绍《文星丛刊》及广播书中爱情观念文字。本轮按台北市级地点入图，不把出版事件直接等同于文星书店具体店址。

## 暂缓入图候选

- 《假如我是女人》后记中的《文星》第68号“1963年6月1日台北出版”和胡秋原控告线索，第一轮只作为文星发表脉络，不单独拆点。

- 中国广播公司介绍《文星丛刊》的节目值得保留，但本轮缺少李敖到场、节目名或录音地点，不新增广播台节点。
