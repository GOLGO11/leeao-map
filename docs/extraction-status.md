# 抽取状态

## 当前取舍

地图不追求把文本中出现的每个地名都放成事件点。当前策略是：

- 主线事件：进入 `events.json`，用于地图和路线播放。
- 地点实体：进入 `places.geojson`，可被一个或多个事件引用。
- 候选地点：先记录在轮次笔记或待复核清单，不立即入图。

这样可以避免地图被细枝末节淹没，同时保留继续深挖的入口。
