# 抽取状态

## 当前取舍

地图不追求把文本中出现的每个地名都放成事件点。当前策略是：

- 主线事件：进入 `events.json`，用于地图和路线播放。
- 地点实体：进入 `places.geojson`，可被一个或多个事件引用。
- 候选地点：先记录在轮次笔记或待复核清单，不立即入图。

这样可以避免地图被细枝末节淹没，同时保留继续深挖的入口。

状态：沉思日记类第二本，初筛与第一轮复查已完成。本书由1958年台大校园零星日记、1959年毕业后从军前日记、1961年四席小屋日记、1966年文星风波断片组成；地点很多，但本书只补强已确认主线，不把校园、饭局、书摊、影院碎片密集入图。

本轮新增入图：

- 本轮不新增独立地图点。原因是核心空间“四席小屋”“碧潭山楼”“凤山预官训练”“文星查禁压力链”均已有主线节点；本书主要提供第一手日记证据和时间校正。

已补强证据链：

- 1959年9月7日起赴凤山接受第八期预官训练。《毕业以后，从军以前》明确日记止于9月6日，并称9月7日以后到凤山受训；本轮将凤山受训节点起点由9月9日校正为9月7日。
- 1961年2月17日租下“四席小屋”。《四席小屋日记》前记说明日记时间为1961年2月16日至6月19日，住在新生南路三段陋巷，并在2月17日记“选中”小房、交下一月房租。
- 1961年6月15日迁到新店“碧潭山楼”。《四席小屋日记》前记说明熬了四个月后决定下乡，在新店选到背山面水的小房；正文又有5月27日看房、6月15日“新汉陪搬家”、6月19日夜始做饭与划船等日记证据。
- 1966年文星封门、移交与《告别文坛十书》压力链。《二十年前的一些断片》记7月27日与文星“封门”、8月2日办移交、11月21日至26日与李中校谈及沿途卖书；本轮补入既有《李敖告别文坛十书》出版与查扣节点的证据链。

暂缓入图候选：

- 1966年11月21日至26日南下沿途卖书路线：台中、彰化、员林、北港、西螺、嘉义、台南、高雄、凤山、屏东、四重溪、恒春、垦丁、鹅銮鼻等地名明确，后续还延伸到台东、知本、花莲、天祥、大禹岭、梨山、新竹等地，路线价值很高；但一次性新增大量城市点会让台湾地图变密，复查后仍作为“禁书与发行网络”路线层候选，不拆成主地图密集点。
- 1961年5月10日赴安坑/碧潭绕山探监，文本称“监所颇大”“隔铁窗与毓田长谈”。复查交叉到“新店安坑监狱/新店监狱”及钟毓田线索，但监所旧址坐标与人物背景仍需专门考证，暂不建点。
- 1961年中央图书馆、美国新闻处、南港史语所、台大各馆舍、傅园、新兴、寿尔康等地名频繁出现，多为读书、饭局、校园生活和私人交游，不进入主线地图。
- 1966年中国大饭店、装订厂、高雄文星分店大勇书店、台中/台南/嘉义书摊等有空间价值，但地点链条复杂，留待“禁书与发行网络”专题层复查。

复查结论：

- 1966年沿途卖书路线适合独立路线层或专题图，不适合直接塞入主线点位。
- 安坑探监线索已有方向，但旧址坐标不足，不进入当前地图。
- 《四席小屋日记》中6月15日“新汉陪搬家”仍是“碧潭山楼”迁居起点；6月19日是安顿后的生活证据，不改变居住起讫日期。

## 《大学日记后期甲集》

状态：沉思日记类第三本，初筛与第一轮复查已完成。本书覆盖1958年6月至12月，地点线索密集，但多数是台大校园、图书馆、饭局、短途游玩与人物交往；本书只收入南港中研院/胡适/史语所资料线和松山机场迎胡适这两个关键节点。

本轮新增入图：

- 1958年6月至12月，南港拜访胡适并使用史语所资料。6月8日李敖赴南港中研院三十周年开放展览并见胡适；7月5日去南港搞徐志摩材料并参观修建中的胡适寓所；11月29日、12月5日、12月17日、12月19日多次赴南港读《清明集》《宋刑统》、送书给胡适、与史语所图书室人员往来。此节点复用 `place-academia-sinica-nangang`，不新增坐标点。
- 1958年11月4日，松山机场迎胡适返台。日记记李敖赴松山机场迎胡适，胡适在人潮中叫出“李敖”，姚从吾随后谈及想请胡适协助李敖论文方向。《李敖自传与回忆》亦重录此条，复查后新增 `place-taipei-songshan-airport` 与独立事件。

已补强证据链：

- 台大后期的胡适影响与论文方向。本书把胡适、姚从吾、徐高阮、史语所资料和宋代婚姻/贞操观念研究放在同一条空间链上，补强既有“台大历史系后期”与“南港中央研究院/史语所”节点。

暂缓入图候选：

- 1958年11月30日赴自由中国社为雷震送《胡适文存二集》，与日后雷震/自由中国线索有关；复查未在本地材料中找到可靠社址，且该条仍属于送书小行程，暂不建点。
- 1958年10月中央图书馆、台大医院、农学院、法院、淡江、圆山饭店前草地、碧潭、台中/东海短行等地名多为校园生活和短途活动，本轮不入主线。

复查结论：

- 南港节点仍保持聚合表达，不拆成胡适寓所、史语所图书室等更细空间；这些具体室内/宿舍位置当前没有必要增加点密度。
- 松山机场迎胡适已由本书与《李敖自传与回忆》交叉确认，且地点清楚、现场重要，正式入图。
- 自由中国社、中央图书馆等候选继续留档，等待后续雷震/自由中国专题或图书馆研究线再处理。

本轮新增入图：
