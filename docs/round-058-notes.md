# 第五十八轮：《李敖访谈录1990-2018》第一轮复查

## 复查目标

本轮复查第五十七轮新增的 1990 年台北地方法庭、1999 年台北国际会议中心两个节点，并核对 2003 年阳明山静养、东丰街/金兰大厦书房、2017 年晚年访谈、2005 年神州文化之旅是否应调整入图。

## 新增与调整

- 新增 `event-2012-2017-jinlan-late-study`。复查确认 2012 年《外滩画报》写李敖搬离居住多年的阳明山书房，住回敦化南路一段公寓，并在访谈中说“我现在住的金兰大厦这个房子”；2014 年《南方周末》写其坐在台北敦南大道金兰大厦书房受访；2017 年《亚洲周刊》又称在台北市敦化南路书房进行此生最后一次媒体访问。此节点补出晚年空间重心从阳明山回到敦化南路金兰大厦。
- 补强 `event-2005-mainland-lectures`。加入 2005 年《中国时报》行前专访与《南方周末》现场侧写，说明凤凰卫视安排、重回北京、北大校办公楼二层演讲报告厅、清华演讲和复旦题词线索。
- 更新 `place-taipei-international-convention-center` 说明：官方资料列地址为台北市信义区信义路五段1号；点位继续按现址近似，具体厅别待考。
- 更新 `place-yangmingshan-qingshan-jianwo-lou` 说明：2003 年 TVBS 的“阳明山住处静养”和 2012 年《外滩画报》的“搬离居住多年的阳明山书房”补强阳明山书房长期性，但没有门牌，暂不新增第二个阳明山点。

## 保留入图

- `event-1990-taipei-district-court-wu-dayou-case` 保留。戴小华跟随李敖前往台北地方法庭，实际到场证据明确；但具体法庭/庭号未见，继续复用 `place-taipei-district-court`。
- `event-1999-new-party-presidential-campaign-ticc` 保留。地点明确为台北国际会议中心；活动日期由 1999 年8月19日报道中的“昨天”推定为 8月18日，复查轮继续标为高可信但保留厅别待考。

## 暂不入图

- 1999 年 Playboy、2000 年《时事开讲》、2001 年《亚洲周刊》、2001 年《广角镜月刊》等多处只称“书房”或“寓所书房”，未给门牌，暂不拆点。
- 2003 年 TVBS 的摄护腺癌术后静养，能补强阳明山住处，但医院名未见，且“阳明山住处”与“青山见我楼”关系仍需跨书核对，不单独建健康事件。
- 2017 年 2月22日台北家中探访和 2017 年 4月《中国报》“台北最昂贵地段”豪宅描写，未给门牌；已由 2017 年2月25日《亚洲周刊》明确的敦化南路书房线索并入金兰大厦晚年书房节点。
