# 第十二轮笔记：《李敖风流自传》复查

## 本轮结论

本轮复查上一轮留下的四个问题：花园新城定位、博爱路保安处与六张犁关系、景美军法看守所旧址坐标、1980年离婚记者会是否入图。复查后新增一个重要事件点，并校正三个旧址坐标。

## 新增入图

- `place-taipei-mainland-building-restaurant`：忠孝东路大陆餐厅 / 大陆大楼。
- `event-1980-hu-yinmeng-divorce-press-conference`：1980年8月28日，李敖在大陆大楼/忠孝东路大陆餐厅记者会宣布与胡茵梦离婚。

入图理由：这是《李敖风流自传》中情感叙事的关键公共节点，有明确日期、地点和李敖本人到场。地点名称有两种文本说法，因此事件 certainty 设为 `medium`，地点保留 `needs_review`。

## 坐标复查

- 花园新城：上一轮坐标偏向新店城区；本轮按“新北市新店区花园一路二段10号”一带校正为社区锚点。萧孟能具体住宅仍未确认，不能写成门牌。
- 警总保安处（博爱路）：本轮以博爱路172号旧址校正。它与《续集》“六张犁看守所”线索不是同一地点，数据中继续保留两个地点，避免混同。
- 景美军法看守所：本轮以白色恐怖景美纪念园区现址“新北市新店区复兴路131号”及“秀朗桥南侧”校正。八号房作为内部空间，不单独设地图点。
- 大陆大楼/大陆餐厅：当前按忠孝东路四段大陆大楼280/290号一带近似；餐厅楼层、是否与大楼同体仍待考。

## 证据复查

- 花园新城初见胡茵梦：除《李敖风流自传》外，《李敖回忆录》同段复述；胡茵梦回忆也称第一次见面在萧孟能花园新城家中。
- 离婚记者会：《李敖风流自传》《李敖回忆录》称“大陆大楼”；《李敖快意恩仇录》《冷眼看台湾》称“忠孝东路大陆餐厅”。本轮合并为同一待考地点。
- 景美军法看守所：《李敖风流自传》《李敖回忆录》称“景美军法看守所”；《快意恩仇录》补出“景美秀朗桥下军法看守所”。

## 暂缓入图

- 台大医院洗肠：有地点但属于私人情感危机的短暂医疗处理，暂不压入主线地图。
- 重庆南路胡开文笔墨庄：更适合未来做“物件记忆/书画流转”专题层，不进入本轮人生轨迹主图。
- 福华大饭店早餐敲定东吴：已作为东吴任教证据背景，不单独拆点。

## 后续待查

- 大陆餐厅当年所在楼层、门牌，以及“大陆大楼”280号/290号门牌关系。
