# 第五十四轮：《李敖书序集续集》第一轮复查

## 复查目标

本轮复查第五十三轮暂缓项，重点核对香港大会堂书展、“香江托孤文件”、鸿禧山庄告发链、陈文茜相关饭局、李世杰《调查局黑牢345天》与李敖出版社地址，判断是否需要新增主图点。

## 复查结论

本轮不新增地图事件、地点或事件-地点关系。《李敖书序集续集》继续作为出版史、查禁、报纸合订本与媒体节目节点的证据补强来源，而不是密集加点来源。

## 数据调整

- 更新 `event-1995-tv-xiaoaojianghu` 摘要：加入1996年10月下旬起，李敖在《笑傲江湖》中与施寄青、陈烨连续三个月十二集解构算命迷信，说明该节目也承接其反迷信、反玄学的公共表达。
- 不新增香港大会堂、台北地方法院、立法院、李敖出版社等点位。

## 暂不入图

- 1965年香港大会堂书展：文本证明“我和文星书店的出版品”在香港大会堂书展占台湾参展册数近九成，适合出版流通专题，但不证明李敖本人到场。
- “香江托孤文件”：可说明下狱前夜规划在香港筹印《李敖选集》等书，但缺少具体可锚定地点。
- 《黄旗梦碎》：李敖同意钱达使用材料告发鸿禧山庄，并安排钱达上《笑傲江湖》节目；台北地方法院与立法院行动主体主要是钱达、刘铭龙、李庆元等人，暂不作为李敖本人足迹。
- 《文茜半生缘》：两次与陈文茜吃饭、彭明敏回台后聚餐等线索无明确地点，且主线权重不足。
- 李敖出版社地址：本书仅能证明李敖出版社出版《调查局研究》《调查局黑牢345天》等书，不能由本书可靠收窄地址。跨书检索见《百家论李敖》中1989年陈漱渝回忆“台北敦化南路490号的李敖出版社”，但该证据应留给未来出版社/报社专题复核，不能直接与金兰大厦住处、白宫大厦《求是报》发行所混同。
