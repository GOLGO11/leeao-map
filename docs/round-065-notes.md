# 第六十五轮：《李敖书简集》初筛

处理时间：2026-05-08

来源范围：`《大李敖全集5.0》（wjm_tcy版）分章节/008.书信函件类/004.李敖书简集/`

## 本轮结论

本轮不新增主线地图点。《李敖书简集》以公开信、狱中秘密通信、政治论战与出版/司法抗辩为主，空间信息多指向收信机关、政论对象或他人行动。对当前地图最有价值的是补强两个已有节点：1981年至1982年第二次入狱，以及《李敖千秋评论》从牢中创刊后的查禁压力。

## 已入库补强

- `event-1981-second-imprisonment`：补入 `006.殷海光·呆·鸟.txt`、`007.环游·还原·抢.txt`、`008.“既联合，又斗争”？.txt`。新增证据覆盖1981年11月23日、11月25日两封“狱中秘密通信”，以及1982年2月11日、14日出狱后揭发台北看守所黑幕的连续公开信。
- `event-1981-1989-qianqiu-review-prison-to-hundred-issues`：补入 `016.保证相互毁灭.txt`，说明《千秋评论》第一期查禁、第十期撕页续售、第十一期《放火·放水·逃》查禁并扣押。

## 重要暂缓候选

- `020.致小金.txt`：1982年10月20日信中明写“我在阳明山神秘兮兮”，但地点只有阳明山泛称，暂不与后来的阳明山“青山见我楼”混同。
- `018.李敖冤狱平反·尤清冤狱在望.txt`：最高法院判决推翻先前高院判决理由，具法律史价值；但未证明李敖本人到最高法院，先列复查候选。
- `001.怎样修理财阀.txt`、`003.财阀有辜.txt`：有中国合成橡胶、重庆南路、财阀诉讼线索，但当前更适合作为法律/商业纠纷专题材料。
- `005.给黄少谷先生的公开信.txt`、`015.给梁肃戎先生的公开信.txt`：收信对象指向司法院、立法院，但不是李敖本人足迹。

## 复查提示

下一轮复查应重点核对三点：

1. 1982年2月10日出狱日期与《书简集》2月11日信中“昨天出狱谈话”的表述是否需要在事件说明中加注。
2. 1982年10月阳明山短住是否能由其他书目交叉到具体住处。
3. “天母静庐的房子”与最高法院判决是否应进入法律专题层，而非主线地图。
