# 第六十六轮：《李敖书简集》第一轮复查

处理时间：2026-05-08

来源范围：`《大李敖全集5.0》（wjm_tcy版）分章节/008.书信函件类/004.李敖书简集/`

## 本轮结论

复查新增一个主线事件，不新增地点。新增事件复用既有 `place-garrison-command-security-office`，用于记录1966年胡适信风波后李敖被警总保安处多次约谈。复查还补强了土城仁爱教育实验所和1981年至1982年第二次入狱两个既有节点。

## 新增入图

- `event-1966-hu-shih-letter-security-office`：1966年11月7日后，徐高阮等人在台北妇女之家举行声讨李敖大会，油印公布李敖给胡适的信，并告到警总；李敖回忆警总保安处由魏以之组长主持，把他约谈多次。本节点只记录李敖实际到场的警总保安处约谈，不把台北妇女之家拆成李敖足迹点。

## 已补强证据链

- `event-1975-tucheng-renai`：`001.怎样修理财阀.txt` 第223行追述1976年6月10日“自台北县土城乡清水村仁爱庄”筹集五万元并邮政划拨，补强李敖在仁爱庄服刑期间仍处理财务事务。
- `event-1981-second-imprisonment`：`009.揭发黑幕以后.txt` 补出一出狱门即被记者缠住、出狱当天下午举行记者会、法务部令各监所戒备、立法院质询与后续舆论反应。

## 复查判断

- 1982年2月10日出狱日期仍按年表与既有跨书证据保留；《书简集》2月11日信中的“昨天出狱谈话”可理解为2月10日出狱后谈话/发表的接续材料，不足以改动出狱日期。
- `020.致小金.txt` 的1982年10月阳明山短住仍不新增地点。文本只给“阳明山”泛称，不足以与后来的阳明山“青山见我楼”合并。
- `018.李敖冤狱平反·尤清冤狱在望.txt` 的最高法院胜诉与“天母静庐”很重要，但仍未证明李敖本人到最高法院；保留为法律专题候选。
- 中国合成橡胶、重庆南路、司法院、立法院等线索主要是公开信对象、机关对象或他人现场，不进入主线地图。
