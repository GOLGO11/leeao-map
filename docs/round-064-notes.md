# 第六十四轮：《李敖书翰集》复查

## 本轮定位

复查第六十三轮留下的不稳定点：雾峰高分院旧址、台北邮政总局、台中地方法院传票、台中市新闻室查书、南部售书路径、台北自来水厂交涉，以及《文星》被封本书内证。结论是本轮不新增地图事件或地点，只校准旧址坐标并补强证据链。

## 调整既有地点

- `place-taiwan-high-court-taichung-branch-wufeng`：坐标由较粗的雾峰旧址近似点校准为中正路738号地址资料近似点 `120.69844, 24.058169`。复查依据显示台中高分院/高分检1963年后在雾峰中正路738号/738之1号址一带办公，1989年迁回台中市自由路；国家文化记忆库的法治街资料指向职务宿舍，不应当作开庭地点。地点仍保留 `needs_review`，因为738与738之1的旧院舍边界还需更细复核。
- `place-taipei-main-post-office-beimen`：不改坐标。复查确认文本附录复函称“台北邮局”，可与台北北门邮局/台北邮局定位大体对应；但具体窗口和大宗函件作业位置仍不可知，继续保留 `needs_review`。

## 补强既有节点

- `event-1965-wenxing-banned`：补入 `009.给殷海光的六封信.txt` 第59、67行。该信在1965年12月26日直接写《文星》连遭两次查扣、一次搜索、一次没收印刷厂稿件后，“今天证实被‘罚’停刊一年”。这比上一轮的香港书展出口压力更直接，作为本书内证补入。
- `event-1965-xinyi-apartment`：补入 `019.为自来水而战.txt`。该文以“本户”代表人身份处理1965年11月水费错误，记三户在建筑上联在一起、自来水厂职员到本户复查、1966年3月2日自来水厂同意按李敖算法收费。它能说明信义路国泰信义公寓迁入后的日常维权现场，但不足以单独列主线事件。
- `event-1966-wufeng-high-court-lawsuit`：摘要补明本轮坐标校准依据。
- `event-1966-taipei-post-office-new-books`：摘要补明附录复函使用“台北邮局”称谓；高雄、恒春仍只是投递抽查地，不拆点。

## 复查后仍不入图

- 台中地方法院传票：`011.我们要像清除恶人一样的清除烂好人.txt` 写请李声庭“代表本人准时出庭”，证明重点是律师代表出庭，不能稳定证明李敖本人到场。
- 台中市新闻室、警总、警局查书小组：`037.软禁中的通信.txt` 证据价值高，但地点不明，且主要是丁颍处理台中书店查扣事；继续作为 `event-1970-house-arrest-surveillance` 的补强，不拆成新地点。
- 南部高雄、台南、嘉义售书和左营天福书局：文本主体为丁颍带《文星》目录南下与书籍流通路径，不是李敖本人足迹；左营只是收书地址。
- 竹南南庄狮头山元光寺：是平景来信处，不是李敖足迹。
- 渝园：`010.一九六五年五月四日.txt` 与 `005.九头鸟之恋.txt` 都出现聚会/饭局线索，但地点旧址未定，主线权重低，暂留候选。
- 台北自来水厂：自来水争议可补强信义路寓所，但核心现场是“本户”，不是李敖到自来水厂办事。

## 本轮结论

本轮不新增事件、地点或事件-地点关系。保留第六十三轮三个新增事件与两个新增地点；校准雾峰高分院旧址坐标，补强《文星》被封与信义路寓所证据。《李敖书翰集》至此完成初筛与第一轮复查。
