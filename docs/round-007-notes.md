# 第七轮笔记：《李敖快意恩仇录》初筛

## 本轮定位

《李敖快意恩仇录》不是单纯年谱式回忆，而是围绕禁书、论战、人物、衙门和自我定性的“快意恩仇”文本。为了避免地图变成论战索引，本轮只收三类地点：李敖本人实际居住点、能改变主线理解的空间节点、以及这本书主题中最典型的冲突现场。

## 本轮新增

- `event-1954-shandao-temple-basement`：台大初期借住善导寺地下室。
- `event-1987-taichung-book-raid-lawsuit`：台中一中宿舍查禁书案与国家赔偿。
- `event-1978-1998-dunhua-south-road-residence`：敦化南路金兰大厦长期住处线索。
- `place-taipei-shandao-temple`：善导寺。
- `place-taipei-dunhua-south-road-residence`：敦化南路金兰大厦住处线索，坐标暂定，需复查。

## 暂缓入图

- 台北、高雄、台中三个市政府被“罚一元”或赔偿故事：本轮保留为法律战候选，不拆成主线点。
- 台中高分院、台中地方法院、最高法院：初筛暂缓；第一轮复查后已补入有出庭描述的台中高分院，其余仍适合将来做“法律战地图”或专题图层。
- 通化街程积宽煤气行地下室：主要是胡虚一困境地点，不是李敖本人路线。
- 台大医学院骨科遗体捐赠设想：属未来安排，不作为实际足迹。

## 复查任务

- 复核善导寺借住与台大入学、法律专修科之间的时间边界。
- 考证敦化南路大厦具体门牌。
- 判断查禁书案是否应扩展为独立法律战图层，包含三市政府、法院与查扣现场。

## 第一轮复查补记

- 新增 `place-taiwan-high-court-taichung-branch`，并加入 `event-1987-taichung-book-raid-lawsuit` 的地点链。理由是文本明确写到台中高分院开庭，且李敖“仍代张桂贞出庭”。
- 敦化南路住处改为“金兰大厦住处线索”。本地全文显示敦化南路相关资料混有金兰大厦、林肯大厦、白宫大厦、出版社、书房与后期住处，不可直接合并为一个精确门牌。
