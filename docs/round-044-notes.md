# 第四十四轮：《李敖五五日记》第一轮复查

## 复查调整

- 《求是报》节点从台北市级定位，收窄到 `place-qiushi-newspaper-dunhua-490`。依据是《你不知道的司法黑暗》所引新闻纸交寄执照：“发行所名称：求是报社文化企业有限公司”“发行所所在地：台北市敦化南路四九〇号十二楼之五”；《李敖放刁集》又说明合约中使用白宫大厦十二楼之五房屋承包《求是报》编辑业务。
- 1990年7月31日、8月7日台中开庭线索，并入既有 `event-1987-taichung-book-raid-lawsuit`，不另造事件。8月8日“台中法院法官来电”可作为该案1990年庭期的日记证据。

## 复查结论

- `event-1991-qiushi-newspaper-launch` 保留为重要出版/媒体节点。新地点代表《求是报》发行所与编辑业务空间，不与金兰住处、林肯会议或李敖出版社十二楼之八混同。
- “中台病不病”复查到《李敖书函集》“在中台医专的讲演”，可确认“中台”指中台医专；但本轮仍未见馆舍、旧址与足够主线权重，暂不入主图。
- 全美华人协会波士顿成就奖继续留海外声援专题候选。文本明确李敖未赴美，以录音代替现场发言，不作为本人足迹。

## 仍暂缓

- 林肯会议地点：五五日记多次写“林肯开会”，全库也有敦化南路610号林肯大厦等线索，但本轮无法证明这些会议就是《求是报》试版/报社地点，不并入本节点。
- 中台医专演讲：适合后续“校园演讲路线”专题层复查。
