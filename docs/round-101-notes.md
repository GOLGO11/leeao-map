# 第一百零一轮：《李敖演讲集》初筛

本轮处理节目演讲类第十一本《李敖演讲集》。全书54篇，时间从1989年到2017年，地点线索很密；初筛策略是不把每篇演讲机械拆点，而是先补主干节点和明确会场。

## 入图结论

- 新增 `event-1989-taiwan-40-gengxin-lecture`：1989年4月14日台北耕莘文教院来台四十周年演讲。
- 新增 `event-1999-taiwan-50-ticc-lecture`：1999年5月9日台北国际会议中心来台五十周年演讲“残山剩水我独行”。
- 更新 `event-1999-2000-challenge-leeao-live-tour`：起点提前到1999年10月7日台湾大学法学院礼堂，并补入高雄技击馆、台南市劳工育乐中心、交通大学、苗栗县立文化中心中正堂。
- 补强既有 `event-1999-new-party-presidential-campaign-ticc`、`event-2005-mainland-lectures`、`event-2010-ntu-lecture`、`event-2011-mainland-lectures`、`event-2014-hong-kong-book-fair`、`event-2017-last-lecture`。

## 新增地点

- `place-taipei-gengxin-cultural-center`：台北耕莘文教院，官方地址辛亥路一段22号。
- `place-kaohsiung-martial-arts-stadium`：高雄中正技击馆，按中正一路96号近似。
- `place-tainan-labor-recreation-center`：台南市劳工育乐中心，按南门路261号近似，坐标待复查。
- `place-miaoli-culture-center-zhongzheng-hall`：苗栗县立文化中心中正堂，按自治路50号近似。
