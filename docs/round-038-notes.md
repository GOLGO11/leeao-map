# 第三十八轮：《一个预备军官的日记》第一轮复查

## 本轮结论

- 不新增主地图点。上一轮新增的高雄仁武第十七师驻地、澎湖白沙大赤崁驻防点均保留，但继续标记 `needs_review`，避免把区域线索伪精确成旧营房或旧民房门牌。
- 仁武驻地证据收紧为：1960年3月3日由步校转入第二军团、第十七师，车过仁武后抵十七师，并分至四十九团四二炮连；3月5日又称“这个地方在仁武附近，凤山北面”。当前坐标只能按仁武范围近似。
- 白沙大赤崁证据收紧为：1961年1月27日至28日部队由仁武出发，经高雄十四号码头登313运输舰，抵马公后到白沙岛东北角大赤崁；日记多次署记“澎湖县白沙乡的大赤崁海边”，并写明驻在赤崁小港口民房。赤崁码头暂并入大赤崁局部空间，不另拆点。
- 高雄十四号码头、马公码头、澎防部看守所均暂缓入主图。前两者留给后续“军旅航线/退伍路线”图层；澎防部看守所可与新店安坑探监线索合并到“政治犯/军中看守所”专题层。
