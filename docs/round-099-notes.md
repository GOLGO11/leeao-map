# 第九十九轮：《笑傲六十年·有话说李敖》初筛

本轮处理节目演讲类第十本《笑傲六十年·有话说李敖》。全书只有六篇，播出日期为2009年8月3日至8月8日，主题是李敖来台六十年的回望。

## 入图结论

- 新增 `event-2009-xiaoao-60-retrospective-fieldwork`，聚合2009年旧地重游与史料拍摄。
- 新增 `place-national-revolutionary-martyrs-shrine-taipei`：国民革命忠烈祠 / 台北忠烈祠。
- 新增 `place-caoshan-guest-house-yangmingshan`：草山行馆（阳明山）。
- 新增 `place-national-palace-museum-taipei`：国立故宫博物院（台北外双溪）。
- 复用 `place-keelung-port` 和 `place-wenxing-bookstore-hengyang`。

## 主要证据

- `001.20090803惨绿少年·初抵台湾.txt` 第5、11行：1949年5月12日从上海到基隆港，2009年5月12日旧地重游基隆港码头。
- `002.20090804狂妄青年·崭露头角.txt` 第51、65行：衡阳路15号文星书店背景，2009年前后到旧址凭吊，旧址内已是永和豆浆店。
- `006.20090808独坐孤岛·创格完人.txt` 第3行：李敖到台北忠烈祠，以拐杖模仿操枪，并在戴笠牌位旁评论。
- `006` 第9行：李敖走到被烧过的草山行馆。
- `006` 第52行：为节目经故宫博物院院长陪同进入库藏，查看《南京条约》。

## 暂缓候选

- 军法处旧址重访：第003篇有国防部带李敖看旧监狱和现场摄影线索，但年份未明，先作为既有军法处节点的候选回证。
- 太原五百完人纪念碑照片：第006篇主要说女朋友在碑前照相，不直接证明本轮李敖本人到场。
- 东丰街书房照片：第005篇出现“东丰街书房里面的一张照片”，仍缺门牌，和《李敖语妙天下》111篇一起待复查。
- 刘长乐“爱之船到基隆”方案：属于未实施方案，实际大陆行仍按既有2005年神州文化之旅节点处理。

## 坐标复核

- 台北忠烈祠：台北市中山区北安路139号，按公开坐标约 `121.532753,25.080458`。
- 草山行馆：台北市北投区湖底路89号，按站点/景点坐标约 `121.539109,25.154632`。
- 台北故宫：台北市士林区至善路二段221号，按文化部开放资料坐标约 `121.548492,25.102357`。
