# 第四十轮：《李敖秘藏日记》第一轮复查

## 修复

- 修复第三十九轮用 PowerShell 管道写入 JSON 时造成的中文问号问题，恢复《李敖秘藏日记》相关事件、地点、证据引文的中文。

## 复查新增

- `event-1985-hualien-taroko-secret-diary`：1985年4月3日至5日花莲、太鲁阁与台中短行。新增 `place-hualien-city-meilun` 与 `place-taroko-gorge`。台中大使大饭店、运动场、忠烈祠、孔庙、来来百货只保留在摘要，不拆成密集点。

## 复查确认

- `place-yuanshan-taiyuan-500-memorial`：公开资料可核到台北市中山区中山北路四段11巷、圆山山脚天文台前方，故从 `area_approx` 调为 `address_approx`，不再标记待考。
- `place-kenting-caesar-park-hotel`：饭店官网地址为屏东县恒春镇垦丁路6号，公开资料亦可互证1986年开业；与李敖1986年9月“开张才半年”叙述相合，故从 `hotel_approx` 调为 `address_approx`，不再标记待考。

## 仍暂缓

- 1986年6月2日郑南榕被捕支援链条：中山分局、法院、土城、马偕医院、南榕家均有空间价值，但“法院”具体机关与部分旧址仍需交叉考证，先留专题候选。
- 1983年装订厂抢书：缺装订厂地址，继续留出版查禁专题候选。
