# 第五十六轮：《李敖对话录》第一轮复查

## 复查目标

本轮复查第五十五轮新增的文星出版权法律战节点，并重新核对 1982 年医院访问、1980 年《台湾时报》谈话会、1986 年香港 Playboy 中文版访谈、龙山寺“五一九绿色行动”讨论和若干日常轶事地点是否足以入主图。

## 复查结论

- 保留 `event-1987-wenxing-rights-litigation` 与 `place-taipei-district-court`。`020.书店升天，作者扫地.txt` 第19行明载朱婉坚 1987 年4月返台，与李敖互商文星复业，并在台北地方法院公证签立七十六年度认字第1505号出版权授与契约；第25、29、73、83至99行继续形成大林、水牛、蔡文甫、於梨华、余光中和文星停业背景的法律战链条。
- 不新增地图点。复查后可确认本书空间线索仍以出版法律战为主，其他访谈场景多缺少可定位地址，或仅属谈论对象空间。
- `place-taipei-district-court` 继续标记 `needs_review`。官方现址资料显示臺灣臺北地方法院院本部位于台北市博爱路131号；现行公证处页面另列新店办公大楼，因此不能把 2026 年可见的公证业务地址直接倒推为 1987 年公证地点。

## 数据调整

- 更新 `place-taipei-district-court` 说明：明确“院本部/博爱路司法机关区域”只是近似锚点，并写入现行公证处地址不可倒推 1987 年旧址的限制。
- 本轮不新增事件、不新增地点、不调整坐标。

## 暂不入图

- `019.《花花公子》访问李敖.txt` 第13行证明香港 Playboy 中文版访问者专程飞往台湾，并在李敖“两千多平方英尺大屋”访问；但原文没有门牌或社区名，暂作未来住处/媒体访谈专题互证，不直接并入金兰大厦。
- `016.五十而不知天命.txt` 第15行提到李敖应《台湾时报》之邀接受座谈式访问，第625行只写“台北的夜”“对坐一堂”，无报社、住处或会场信息，暂不落点。
- `017.只有作战而死，怎能求情而死.txt` 与 `019.《花花公子》访问李敖.txt` 所涉龙山寺“五一九绿色行动”是党外运动讨论，未见李敖本人到场。
- 1982年3月16日多篇访问中的医院语境未给医院名；“金蛋糕”“新高原药房”“仁爱路远东百货楼上餐厅”等生活轶事地点，当前主线权重不足，继续留作日常地名候选。
