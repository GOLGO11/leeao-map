# 第九十四轮：《李敖有话说》第一轮复查

## 复查重点

本轮承接第九十三轮初筛，重点复查三项：

- 后段外景是否还有实际到场点遗漏。
- 凤凰卫视《李敖有话说》台北录影空间能否从文本收窄。
- 党史馆系列是否需要从单点拆成多事件或多地点。

## 新增入图

复查补入阳明山阎锡山故居/墓园：

- `683.2006-10-18：采访阎锡山公墓.txt` 第 3 行给出“永公路245巷34弄”一组门牌，并明确说“今天我做一次外场，我到了台北市的近郊阳明山”；第 5 行说明此处是阎锡山晚年在台湾的家；第 11 行由张日明带李敖先到墓园；第 75 行走到阎先生故居门口；第 167 行说明本集介绍概况，后续还会讲解为什么到阳明山介绍阎先生故居。
- `684.2006-10-19：谈山西王阎锡山.txt` 第 3 行说明李敖在“中国台湾台北，阳明山上的一个奇怪的房屋里面录影”；第 17 行说明这间房是阎锡山五间房中的书房；第 25 行说明阎锡山最后十年在这个房间著书立说。
- `685.2006-10-20：只有共产党能够斗垮阎锡山.txt` 第 3、5 行继续说明李敖在阎锡山故居房屋内现场做节目，并回顾前两集到阎锡山坟前。
- `686.2006-10-23：山西王阎锡山.txt` 第 3、15、25 行再次说明李敖坐在阎锡山房间/椅子旁谈阎锡山，并点出圆山饭店前五百完人招魂冢只是谈论对象，不是本轮到场点。

新增地点：

- `place-yan-xishan-residence-tomb-yangmingshan`：阎锡山故居 / 阎锡山墓（阳明山）。地址按台北市文化局公开资料复核为台北市士林区永公路245巷34弄273、277号；墓园按同巷弄近似处理。

更新事件：

- `event-2006-youhuashuo-cultural-field-trips`：外景路线从四站扩为五站，即党史馆、阎锡山故居/墓园、林语堂故居、钱穆故居、淡水红毛城。

## 复查结论

- `616` 至 `620` 是党史馆史料库现场延续，仍合并在党史馆外景路线，不拆成多事件。
- `727.2006-12-19：我的六点真相.txt` 只说明“2006年12月12号，在中国台湾的台北录影”，能强化台北录影事实，但不能定位到具体棚别、门牌或制作公司。
- 节目中的总统府、立法院、圆山饭店前五百完人招魂冢、北京/海南岛等，均为谈论对象、候选构想或既有主线，不在本轮新增。
