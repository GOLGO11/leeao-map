# 第六十八轮：《李敖书札集》第一轮复查

处理时间：2026-05-08

来源范围：`《大李敖全集5.0》（wjm_tcy版）分章节/008.书信函件类/005.李敖书札集/`

## 本轮结论

复查新增一个事件，不新增地点。新增事件为1970年9月4日凌晨安坑监狱外接雷震出狱，复用上一轮新增的 `place-xindian-military-prison-ankeng`。上一轮新增的安坑探监事件与四季出版公司查扣事件均保留。

## 新增入图

- `event-1970-ankeng-prison-lei-chen-release`：李敖在长期跟监状态下，约《纽约时报》兼 Time-Life 记者 Donald H. Shapiro 与美联社记者 Leonard Pratt 到新店安坑监狱接雷震出狱，试图突破官方封锁。`《李敖情书集》` 明写“九月四号半夜”，`《李敖书札集》` 又以雷美琳为对象回忆1970年9月3日约外国记者接雷震出狱；本轮按雷震出狱通说日期1970年9月4日凌晨入图。

## 复查调整

- `place-xindian-military-prison-ankeng` 坐标校准为国家文化记忆库所列 `121.5235786, 24.9573913`。
- `event-1970-house-arrest-surveillance` 摘要中“安坑监狱旧址仍待专门考证”改为“该安坑行动已另列节点”，避免与新地点状态矛盾。

## 继续暂缓

- 四季出版公司节点保留。统一装订厂、台北联丰书报社、桂冠图书公司虽然出现在查禁链条中，但暂无可靠旧址，不拆点。
- 殷海光后事线索涉及台大医院解剖室、太平间、怀恩堂、市立殡仪馆和玉成街，价值高，但本轮判断更适合人物专题层。
- 信义路派出所户口校正、林肯大厦应召站、最高法院平反与天母静庐、林义雄家属回台、彰化警察枪击案等继续留作专题候选。
