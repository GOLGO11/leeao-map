# Round 023 Notes - 《李敖文存二集》第一轮

## 初筛范围

本轮处理 `002.精品散文类/005.李敖文存二集/` 全书 16 篇正文与目录。此书一方面收 1979 年复出后的报刊文章，另一方面重收 1960 年代旧文、书信与后记，所以初筛重点放在出版、办刊、广告拒登、住处/牢狱回忆等可与人生轨迹互证的线索。

## 本轮新增

- 新增 `event-1967-wenfeng-magazine`：李敖隐名参与《文风》杂志创办，一期即受压停办。地点暂用 `place-taipei`，因为本书没有给出编辑、发行或印刷门牌。

## 补强既有节点

- `event-1966-farewell-ten-books-censorship`：补入《一封存证信的故事》，作为《中央日报》先收《李敖告别文坛十书》广告费、后拒登广告的证据；同篇又说 1979 年复出时忠佑公司送广告稿到《中央日报》仍被拒。

## 暂缓地点

- 罗斯福路改名争论、中国小姐选拔、大专联考作文题、张小兰案：这些是李敖 1979 年复出后的公共评论对象空间，不是本人足迹，暂不入图。
- 《过早的答案》中的台大融融社、《中央日报》短评、《文星》编务辞任：可补强 1964 至 1965 年文星/台大论战背景，但初筛不拆新点。
- 《眼看黑暗到来，又目送黑暗归去》中的“文星分家”“豪华监狱”：疑似可与信义路寓所及《文星》资料室并读，但本文本未给门牌，不新增地点。
- 《论没有“流血的自由”》仅署“1967年...在台北写”，能补写作状态，暂不单独入图。

## 待复查

- 《办杂志的一些意愿》后记作“1977年7月”，但《李敖回忆录》《李敖快意恩仇录》《李敖风流自传》等多处互证《文风》事件为 1967 年。本轮先按 1967 年入图，复查时继续核对版本差异。
- 继续寻找《文风》杂志登记资料、编辑部或发行地址；如只能确认“编辑部在李敖家里”，仍需判断是否可与信义路国泰信义公寓合并。
- 继续核对《中央日报》1966、1979 年办公地址，避免把报社地点伪精确化。
