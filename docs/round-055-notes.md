# 第五十五轮：《李敖对话录》初筛

## 本轮定位

《李敖对话录》收1982年至1987年间部分对话，材料多为思想、政治、文化论战与媒体访谈。初筛策略是：不把谈论对象地名、历史比喻地名和访谈者所在地都放入主图，只收能改变李敖空间主线理解的到场、法律、出版或媒体节点。

## 新增入图

- `event-1987-wenxing-rights-litigation`：1987年4月至6月，文星出版权公证与版权诉讼。`020.书店升天，作者扫地.txt` 记朱婉坚4月返台，与李敖互商文星复业，并在台北地方法院公证签立出版权授与契约；4月17日李敖亲自与大林、水牛方面接洽，随后围绕蔡文甫、於梨华、余光中等旧文星作者再出版行为展开告发与诉讼。
- `place-taipei-district-court`：台北地方法院。当前按台北地方法院/博爱路司法机关区域近似，标记 `needs_review`；复查轮需核1987年公证处与审判庭具体旧址。

## 证据补强

- 本轮把1960年代《文星》被封、文星书店结束后的出版权益，延伸到1987年文星复业/版权法律战，形成“文星出版空间 -> 被封 -> 权利追索”的后续链条。
- `019.《花花公子》访问李敖.txt` 证明1986年香港 Playboy 中文版专程飞往台湾，在李敖“大屋”访问他，可作为未来“海外媒体访谈/金兰住处”互证材料；但本轮不直接以“大屋”推定门牌。

## 暂不入图

- 1982年3月16日多篇访问：多为医院、性格、思想与政治问答，虽有“医院考验人性”语境，但未给具体医院名。
- `003.在台大·在法庭.txt`：台大与法庭更多是回忆/论述场景，未给具体诉讼地点。
- `016.五十而不知天命.txt`：1980年《台湾时报》谈话会有“台北的夜”“对坐一堂”，并有胡茵梦在场，但没有明确访谈地点。
- `017.只有作战而死，怎能求情而死.txt` 与 `019.《花花公子》访问李敖.txt` 中的龙山寺“五一九绿色行动”属于党外群众运动讨论，当前未见李敖本人到场。
- “金蛋糕”“新高原药房”“仁爱路远东百货楼上餐厅”等生活轶事地点坐标与主线权重不足，留作日常地名候选。
