# 第五十二轮：《李敖书序集》第一轮复查

## 复查目标

- 回查第五十一轮暂缓候选：唐德刚来访、李世杰家、罗开甲处、牯岭街书摊、第一大饭店喜临门、机场送居浩然，以及《历年办理匪案汇编》白色恐怖空间线索。

## 本轮结论

本轮不新增地图事件或地点，继续保持《李敖书序集》不扩密的判断。

复查后确认：这本书的价值主要是补强既有牢狱线与出版史线索，而不是新增人生路线点。多数地点仍属于被介绍书籍的历史舞台、政治案件对象空间或短暂社交/书摊记忆。

## 暂不入图

- 唐德刚来访签约《李宗仁回忆录》：已经作为金兰大厦住处证据补强；不另做独立地图事件，以免把出版签约碎片推得太密。
- 1986年6月30日亲赴李世杰家：跨库可见《白色恐怖述奇》也写李敖亲赴其家、说动写作，但仍未见地址。
