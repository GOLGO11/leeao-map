# 第一百零五轮：《快意还乡——李敖神州文化之旅》初筛

日期：2026-05-09

## 本轮定位

《快意还乡——李敖神州文化之旅》位于 `010.节目演讲类/013.快意还乡——李敖神州文化之旅/`，共12篇逐日记录，覆盖2005年9月19日至9月30日台北、香港、北京、上海再返香港/台北的行程。初筛原则是：只新增能支撑2005年回乡主线的明确到场节点；航班、中转、饭局、棚内聊天、临时景点和媒体采访背景先不拆成主图点。

## 新增事件

- `event-2005-shenzhou-tiananmen-palace-museum`：2005年9月20日登天安门、参观北京故宫。`002.9月20日神州文化之旅第二天.txt` 第5、19、21、45、47、167行可串起从钓鱼台出发、登上天安门、在故宫太和殿前现身、参观故宫并公开修正旧评。
- `event-2005-shanghai-shidong-middle-school-return`：2005年9月27日拜访上海母校市东中学。`009.9月27日神州文化之旅第九天.txt` 第5、21、29、31行明确写李敖拜访缉规中学（现上海市东中学）、在校礼堂讲话并题字。
- `event-2005-hong-kong-cai-yuanpei-tomb-visit`：2005年9月29日香港凭吊蔡元培墓。`011.9月29日李敖神州文化之旅第十一天.txt` 第7、15、17、19行记录李敖到香港仔华人永远坟场蔡元培墓前献花拜祭，并现场考订碑文与蔡元培历史。

## 新增与校正地点

- `place-tiananmen-beijing`：天安门城楼。坐标按维基共享资源所列物体位置近似。
- `place-palace-museum-beijing`：故宫博物院（北京）。坐标按北京市文物局所列景山前街4号院区近似。
- `place-cai-yuanpei-tomb-aberdeen-hong-kong`：蔡元培墓（香港仔华人永远坟场）。外部资料可确认墓在香港仔华人永远坟场；当前坐标只落到坟场山坡区域，具体墓位保留待考。
- `place-shanghai-jigui-middle-school`：从旧的无坐标“上海市缉椝中学”校正为“上海市市东中学 / 缉椝中学旧址”，坐标按市东中学/上海市市东实验学校校园近似。

## 补强既有节点

- `event-2005-mainland-cultural-revisit`：补入本书第001篇第11、91、123行和第012篇第45、47、51行，增强2005年神州文化之旅重回北京、新鲜胡同小学与钓鱼台行程链。
- `event-2005-mainland-lectures`：补入第003篇北大校办公楼二楼演讲报告厅、第005篇清华中央主楼报告厅和主楼贵宾室、第008篇复旦逸夫科技楼与校史馆证据。

- 首都机场、香港机场、浦东机场、磁浮列车、钓鱼台国宾馆、老锦江饭店、香港海逸酒店等均有行程价值，但本轮只作为路线背景或既有城市节点补充，不单独拆点。

- 朱家角、黄浦江外滩、太平山、香港游艇和珍宝海鲜舫属于临时游览或饭局层级，权重低于本轮三条主线，留待复查轮再判定。
