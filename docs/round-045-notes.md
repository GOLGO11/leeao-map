# 第四十五轮：《李敖随写录前集》初筛

## 本轮判断

- 《李敖随写录前集》是短札集合，不是连续行旅日记。大量地名只是新闻对象、评论对象或日常片段，本轮不做密集拆点。
- 主线价值最高的是《求是报》停刊后的出版转向：从日报经营退回到《李敖求是评论》以及“大书”写作。
- 本人到场线索中，野柳两次出现，且有题名、日期、动作和景观描写，保留为本书的代表性空间记忆。

## 本轮新增

- `event-1991-qiushi-newspaper-end-review-magazine`：1991年《求是报》8月停刊，11月改办《李敖求是评论杂志》，1992年4月1日第六期后停刊。地点沿用 `place-qiushi-newspaper-dunhua-490`。
- `event-1991-yeliu-wangyuan`：1991年9月1日野柳望远，9月7日与陈又亮再游野柳并登灯塔。新增 `place-yeliu-geopark`，坐标按野柳地质公园/野柳岬近似。

## 暂缓候选

- 淡水“四海一家”与看海：1991年10月13日本人到场明确，但当前更像日常短札，地点还需复查。
- 荣民总医院三日：连续到场与老兵空间观察有价值，但主线权重暂不足。
- 故宫博物院购书、牯岭街书店、胡秋原新店中央新村住宅假扣押：先分别留作图书/法律战专题候选。
- 华视、中视、东海大学邀约均为谢绝，不入足迹地图。
