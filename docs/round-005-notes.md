# 第五轮笔记：《李敖回忆录》初筛

## 本轮定位

《李敖回忆录》本身就是人生路线的分章重述，和现有地图主线高度重合。本轮不把重复叙述拆成新的密集点位，而是把它作为“主线证据总复核”的第一步：能直接支持既有事件的，补入 `source_refs`；只有能补出新叙事节点的，才新增事件。

## 本轮入图与补证

- 为哈尔滨、北京/太原、上海、台中、台大、军队、山居、文星、软禁、监狱、二进宫、东吴、电视阶段补入《李敖回忆录》证据。
- 新增 `event-1997-cathay-hospital-memoir-surgery`：1997年3月31日，国泰医院手术前写定《李敖回忆录》。
- 新增 `place-cathay-general-hospital-taipei`：国泰综合医院（台北），坐标暂按台北院区处理，需复查。

## 暂缓入图

- 杭州南路文献会：章节明确写到住进杭州南路文献会，初筛先作为复查候选；第一轮复查已用《李敖自传与回忆》“仁爱路2段9巷27号”补强地址，正式入图。
- 胡须张、福华大饭店、太平洋崇光百货：与东吴任教促成过程有关，但主线地图价值低于东吴大学本身。
- 太原公园、香厂、榆次、太谷等早年短途记忆：暂作太原段背景，不独立入图。
- 他人祖籍、父辈求学地、亲友经历中的地点：除非直接改变李敖本人足迹，否则不混入主路线。

## 复查任务

- 逐章复核是否遗漏“本人到场且改变主线理解”的地点。
- 考证杭州南路文献会旧址、国泰医院具体院区、东吴任教校区、电视录影地点。
- 核对牢狱地点旧称与现名，特别是警总保安处、景美军法看守所、土城仁爱教育实验所、台北看守所之间的对应关系。

## 第一轮复查补记

- 新增 `event-1962-wenxianhui-hangzhou`：1962年2月1日至1963年5月前后入文献会并住杭州南路办公处。
- 新增 `place-taipei-wenxianhui-renai-hangzhou`：开国五十年文献会（仁爱路二段9巷27号）。坐标按仁爱路二段9巷与杭州南路一带暂定，继续标记 `needs_review`。
- 复查后仍暂缓胡须张、福华大饭店、太平洋崇光百货、真相新闻网/TVBS录影棚等地点；它们多为东吴任教或电视节目背景，尚未改变主线地图结构。
