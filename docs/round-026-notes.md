# 第二十六轮：《波波颂》复查

处理日期：2026-05-07

## 复查问题

本轮围绕第二十五轮留下的三个问题复查：

- 1986年大安分局能否收窄到旧址或门牌。
- 1990年《波波颂》出版能否建立地图节点。
- 1973-1974年“狱中作”是否应继续并入第一次政治犯时期事件。

## 复查结论

- 大安分局节点保留。全库可见大安分局在李敖软禁、跟监、撞车赔偿、林肯大厦检举等材料中多次出现，但未见1986年这次笔录事件的门牌或旧址；因此 `place-taipei-daan-police-precinct` 继续按 `district_approx` 处理，并保留 `needs_review`。
- 《波波颂》出版节点暂缓。自序署1990年9月25日，《李敖五五日记》可补出1990年10月出书事实，但本地材料未收窄到出版/发行地点，暂不新增主线地图点。
- “狱中作”证据补全。`001` 至 `007` 连续七篇均署1973年11月至1974年7月“狱中作”，与既有第一次政治犯时期重合；复查后全部作为 `event-1972-1975-military-law-room-8` 的来源证据，不拆密集重复点。

## 暂缓候选

- 波斯猫“波波”与书桌生活史：有住处日常价值，但地点仍未明；《李敖回忆录》《李敖风流自传》和节目转录对“花莲/宜兰”有版本差异。
- 大安分局的其他材料：软禁跟监、撞车赔偿、林肯大厦检举等更适合以后做“警察/法律战专题层”，本轮不并入1986年笔录事件。
- 国宾饭店、嘉义/板桥跑步活动、台北地方法院多案：仍按评论对象或法律背景处理，不进入主线地图。
