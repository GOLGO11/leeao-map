# 第九十七轮：《李敖语妙天下》初筛

本轮处理节目演讲类第九本《李敖语妙天下》。目录列2009年1月1日至6月12日节目文字稿113篇，并说明112集后半即113集，47、57、63、64、72集有缺分钟或不完整情况。

## 入图结论

- 新增 `event-2009-tv-leeao-yumiao-tianxia`，表示2009年《李敖语妙天下》重返电视讲坛。
- 新增 `place-gaodian-tv-yumiao-tianxia-taipei`，作为高点电视台 / 《李敖语妙天下》录影播出空间的市级暂置点。
- 同一事件挂接 `place-yangmingshan-qingshan-jianwo-lou`，因为文本直接证明李敖从阳明山工作间下山录影。

## 主要证据

- `001.2009-01-01：为什么重返萤光幕？.txt` 第13、15行：郑乃嘉、佳思比媒体行销安排节目，从新加坡作为起点，面向22亿华人人口。
- `007.2009-01-09：许历农才是觉悟了的国民党.txt` 第3行：李敖说“刚刚我从阳明山下来录影”，并说明在台北阳明山有一间工作间，每次录影由公司专车接下山。
- `075.2009-04-22：录影当天是二二八.txt` 第3、5行：节目为录影后制，繁体字幕给台湾高点电视台，简体字幕给以新加坡为主的华人观众，现场有工作人员。
- `074.2009-04-21：我家凶悍的老太(四).txt` 第25行：阳明山书房开窗可望见母亲灵骨塔，补强阳明山书房空间记忆。

## 暂缓候选

- 东丰街二楼书房：`111.2009-06-11：演艺人员的毛病出在哪？.txt` 第3行提到台北市东丰街书房、蔡牙科楼上，但没有门牌，需复查照片、牙科名录或其他书信材料。
- 高点电视台当时地址：外部线索有馆前路、重庆南路、新北土城等冲突，本轮不下门牌坐标。
- 佳思比媒体行销：文本证明制作安排，但未给地址，待复查。
- `106.2009-06-04：不须放屁，试看天地翻覆.txt` 展示家中书房与收藏，但未给地址，暂不新增点。

## 方法备注

《李敖语妙天下》大量地名来自历史、政治、文学与新闻评论。`113.2009-06-12：身临其境太笨了.txt` 还特别强调李敖的“卧游”方法，因此本轮只收录能证明李敖本人实际准备、录影、制作或播出链条的地点，不把讲述对象地名混入足迹。
