# 第八十六轮：《李敖秘密书房》第一轮复查

## 复查范围

- 复核第八十五轮新增的《李敖秘密书房》家中书房录制节点。
- 重点回看当前二楼书房、敦化南路十二楼书房、阳明山“四个窝”、新党义卖/书展、励馨基金会义卖和杨西崑来访线索。

## 本轮结论

- 保留 `event-1999-2000-tv-secret-study-at-home` 与 `place-leeao-secret-study-taipei`，不新增地点。
- 日期标签由“1999年底至2000年5月5日”校正为“约1999年12月至2000年5月5日”。`003.书房搜秘.txt` 第3行以1999年12月14日前后的巴拿马运河新闻为引子，可证明节目已进入1999年12月语境，但不能反推第一集精确日期。
- `001.秘密书房泄密了.txt` 第7行的“敦化南路一个十二楼的书房”补强敦化南路十二楼书房线索，但同句明确说“不是现在这个书房，现在这个书房是二楼”，因此不能把节目书房并入金兰大厦。
- `090.20000412万书如海一身藏（一）.txt` 第87行“我有四个窝，还有一个在阳明山”补强阳明山书房/山居说明，但不证明节目在阳明山录制。

## 继续暂缓

- `005.我的压箱宝.txt` 与 `103.20000501李敖六五（四）.txt` 证明李敖为新党总统选举进行义卖、捐出收藏品，甚至提到新党党部，但未给义卖会场；暂不入主图。
- `006.印第安式礼物.txt` 提到把对联捐给励馨基金会义卖，属于物件流转/慈善义卖线索，缺会场，不建点。
- `008.怀想杨西崑.txt` 的台北宾馆、松江路家中饭局、外交部讲习所和杨西崑到李敖家来访，属于私人交往与人物回忆，时间和门牌均不足，暂列候选。
