# Round 017 Notes - 《传统下的再白》第一轮

## 处理范围

本轮处理 `002.精品散文类/002.传统下的再白/` 全书 8 个正文文件和目录。由于本书以爱情、女性、知识人、日记和王世杰材料为主，地名大量属于论述对象或被批评对象空间，本轮继续沿用精品散文类的严格取舍。

## 本轮新增入图

- 1966年11月5日前后，《李敖告别文坛十书》出版与查扣。本轮把它作为《文星》被封后、李敖早期著作遭围堵的聚合节点处理。
- 《传统下的再白》本身不单独作为出版节点入图，而是纳入“告别文坛十书之六”的证据链。

## 证据依据

- 《李敖自传与回忆》称 1966 年多种著作被查禁，11月5日出版《李敖告别文坛十书》，并在装订厂被治安人员抢走。
- 《李敖著作查禁清单及官方理由》称《传统下的再白》为“李敖告别文坛十书之六”。
- 《日记八则》记录王世杰要求删去《蒋廷黻选集》李敖序未果，最后经人转手将序割除；李敖随后把该序重新发表在《李敖告别文坛十书》里。

## 暂缓入图候选

- `008.王世杰“蒙混舞弊”的铁证.txt` 中的长安东路一段二十八号、仁爱路四段十二巷八弄八号、中央研究院等地点，是王世杰及中研院行政腐败指控的对象空间，不是李敖本人足迹。本轮不入主线地图。
- `007.日记八则.txt` 中“与李声庭吃饭”缺地点，暂不入图。
- `006.怪定义四则.txt` 署记“1960年12月1日作于军中”，缺具体营区；已有军队主线节点，不重复拆点。
- `003.李敖先生谈女人（语录）.txt` 中“旗袍店”仅为短暂见闻，缺店名与地点，不入图。

- 评估王世杰、中研院、李敖与《蒋廷黻选集》序文割除事件是否另建“中研院论战/出版审查专题层”。
