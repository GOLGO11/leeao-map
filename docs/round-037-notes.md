# 第三十七轮初筛记录：《一个预备军官的日记》

## 处理范围

- `006.沉思日记类/005.一个预备军官的日记/`
- 本轮重点处理引言、凤山步校起讫、仁武第十七师驻地、澎湖白沙大赤崁驻防与退伍返台中证据。

## 入图结论

- 校正 `event-1959-fengshan-training`：受训时间收束为1959年9月7日至1960年2月25日。2月25日第三梯次毕业典礼后授阶，午后宣布分至第二军团。
- 新增 `event-1960-renwu-17th-division`：1960年3月3日至1961年1月26日，高雄仁武第十七师/四十九团四二炮连驻地。
- 新增 `event-1961-baisha-dachikan-penghu-service`：1961年1月27日至2月6日，澎湖白沙大赤崁驻防。
- 细化 `event-1961-penghu-discharge`：补入2月5日移交、2月6日马公码头领退伍证、乘三一一军舰抵高雄的日记证据。

## 新增地点

- `place-renwu-17th-division-camp`：高雄县仁武乡第十七师驻地线索。具体营房旧址待考，坐标按仁武范围近似。
- `place-penghu-baisha-dachikan`：澎湖白沙乡大赤崁。具体民房与码头旧址待考，坐标按村落/港口近似。

## 关键证据

- 1959年9月7日，日记开始于后壁车站，记从台中经彰化南下，傍晚抵凤山。
- 1960年2月25日，参加第三梯次毕业典礼，午后宣布分至第二军团。
- 1960年3月3日，自五块厝转入第二军团，十八人分到十七师；车过仁武，抵十七师，分至四十九团四二炮连。
- 1961年1月26日至28日，部队离仁武，经高雄十四号码头上运输舰，抵马公后到白沙岛东北的大赤崁。
- 1961年2月6日，在马公码头领退伍证，三一一军舰8时开，告别马公、澎湖。
