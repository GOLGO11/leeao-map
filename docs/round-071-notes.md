# 第七十一轮：《李敖书牍集》初筛

处理时间：2026-05-08

来源范围：`《大李敖全集5.0》（wjm_tcy版）分章节/008.书信函件类/007.李敖书牍集/`

## 本轮原则

本书是1986年9月至1987年8月间的书牍与公开信，地点多为收信机关、被批评对象或报刊引文。初筛只收三类材料：明确的查禁现场、李敖亲自进入或反复使用的法律空间、能补强既有主线事件的硬证据。

## 新增入图

- `event-1986-guting-roosevelt-road-book-seizure` / `place-taipei-guting-roosevelt-road-area`：据 `005.贵市长言而无信，令人羞羞.txt`，1986年10月28日下午，台北市警察局古亭分局罗斯福路派出所警员胡鹏翔在马路上查扣《万岁评论丛书》第三十一至三十五期及《新观点丛书》共四十一册。因文本未给具体路口或派出所门牌，本轮仅按古亭/罗斯福路辖区近似，保留 `needs_review`。
- `event-1986-1987-taipei-court-libel-cases`：据 `021.司法院有人在打鼾吧？.txt`、`024.这是什么问案标准？.txt` 与 `027.从恐吓到通缉.txt`，李敖1986年1月4日在台北地方法院自诉蔡仁坚、林正杰、吴祥辉、汪梦湘诽谤案，1987年2月连续向司法院和台北地院院长抗议案件延宕；同年4月又记吴祥辉因《李敖死了》诽谤案传唤不到而遭台北地方法院通缉。地点复用 `place-taipei-district-court`。

## 已补强证据链

- `event-1986-1987-chiang-research-publication-censorship`：补入 `006.“惭愧公荣文王”.txt` 中1986年10月31日秘密上版《蒋介石研究》的内证，以及 `038.给吴三连的预告信.txt` 中《蒋经国死了》/《蒋经国研究》寄送吴三连的线索。
- `event-1986-daan-precinct-report`：补入 `027.从恐吓到通缉.txt`，确认吴祥辉恐吓案由信义路派出所、大安分局侦办移送，并由台北地检处以恐吓罪嫌起诉。
- `event-1987-taichung-book-raid-lawsuit`：补入 `037.盗贼政府.txt`，作为台中母亲住所与台北、台中三处藏书处被搜查、李敖赶回台中处理的本书内证。

## 暂缓候选

- `030.答非所问是不行的.txt` 的台北市政府新闻处/《文星杂志》注销行政交涉，李敖确有亲送函件，但更像文星行政战专题，暂不另建市府新闻处点。
- `026.警察头子们是什么意思？.txt` 的王剑芬外侨居留、新店花园新城地址，主要是被检举对象空间，不是李敖足迹。
- `033.寄刘辰旦谈党外内史.txt` 的台中探丁颍、车站归途，地点清楚度与主线权重不足；同文立法院前抗议《国安法》是李敖在计程车中听广播得知，不能作为到场节点。
- `034.给李政一的一封信.txt` 的刘辰旦高雄书画展序，文本能证明写序，不能证明李敖到高雄。

## 下一轮复查重点

- 复核古亭分局罗斯福路派出所旧址或辖区边界，避免把“警员所属机关”误标为“查扣发生门牌”。
- 复核台北地方法院1986年至1987年院址/庭址，与现博爱路坐标是否可继续作为近似点。
- 再查《文星杂志》注销、《自立晚报》/吴三连诽谤案是否应从法律专题层拆出独立行政或媒体节点。
