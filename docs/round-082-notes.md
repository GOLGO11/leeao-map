# 第八十二轮：《李敖笑傲江湖》第一轮复查
处理时间：2026-05-09

来源范围：`《大李敖全集5.0》（wjm_tcy版）分章节/010.节目演讲类/001.李敖笑傲江湖/`

## 复查原则

本轮复查第八十一轮留下的四个问题：真相新闻网旧址、录影棚位置、1995年10月30日开播证据和节目后期平台变化。复查结论是：本书能显著补强“真相新闻网电视台”与节目制作方式，但仍不能收窄到门牌或棚别；因此不新增地图点，只增强 `event-1995-tv-xiaoaojianghu` 和 `place-truth-news-network-taipei` 的证据链。

## 补强证据链

- `024.李敖的三个主义.txt`：称李敖与真相新闻网合作做《笑傲江湖》节目，合约快满一年；又称真相新闻网有雅量请他来“台里面”做节目，并说自己在真相新闻网主持节目。
- `028.何其太雅.txt`：说明节目制作细节，包括字幕、三台摄影机、三位工作人员和不得片面删改节目的合约条款；又称“目前的真相新闻网电视台”。
- `113.李敖快意恩仇录（一）.txt`：称李敖只有在真相新闻网、在周荃和周菲主持的电视台里，才有这种言论自由。
- `115.李敖快意恩仇录（三）.txt`：称《笑傲江湖》讲下来七百多场，每集半小时，字幕平均约6500字，可补充节目规模和文本化价值。

## 继续暂缓

- 真相新闻网旧址：本书没有门牌、楼层、街道或录影棚名称，继续按台北市级坐标处理。
- 录影棚位置：第28集只说明录影方式，不给地址；不把“台里面”推定为具体棚别。
- 1995年10月30日开播：本书第一集可证明节目开端和10月20日签约，但没有直接写“10月30日开播”；日期仍以年表和既有证据为准。
- 后期平台变化：本书中未见足以证明节目后期转移平台的地点线索；第115集“七百多场”只补规模，不拆新节点。
- 来来饭店日本料理：仍作为签约谈判背景，不单独建点。
- 吕安妮事件记者会、台湾大学商研所会面、福华饭店旁受伤线索，均更适合未来“吕安妮事件/教育争议”专题层。

## 复查结论

《李敖笑傲江湖》第一轮复查完成。保留 `place-truth-news-network-taipei`，不提高坐标精度；保留 `event-1995-tv-xiaoaojianghu`，不新增事件。当前判断：本书对地图的主要价值是把李敖1995年后电视阶段从泛台北收束到“真相新闻网电视台”这个媒体空间，而不是提供可精确定位的录影棚。
